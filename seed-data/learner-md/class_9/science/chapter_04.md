## 🌍 Introduction — Motion Is Everywhere

Everything in nature is in motion. Large astronomical objects move. Tiny subatomic particles move. Butterflies flit, snakes slither, hares hop, horses gallop, dust particles dance in sunlight, smoke particles move in air, tides rise and fall, and clouds gather.

The world has many kinds of motion. Some motion is simple; some is very complex.

Scientists often begin by studying simplified forms of complex phenomena. In motion, useful simplified forms include:

| Type of Motion | Simple Meaning | Example |
|---|---|---|
| **Linear motion** | Motion along a straight line | Train on a straight track |
| **Circular motion** | Motion along a circular path | Stone tied to a string and whirled |
| **Oscillatory motion** | To-and-fro motion about a mean position | Swing, pendulum |

This chapter focuses mainly on:

| Focus Area | What You Will Learn |
|---|---|
| **Motion in a straight line** | Position, distance, displacement, speed, velocity, acceleration |
| **Motion graphs** | Position-time and velocity-time graphs |
| **Kinematic equations** | Equations for constant acceleration |
| **Uniform circular motion** | Constant speed but changing velocity direction |

> [!TIP]
> Motion is not studied only by looking at objects. It is described using **measurements**, **graphs**, and **mathematical relationships**.

### Core Questions

| # | Question |
|---|---|
| **1** | How much distance should we maintain from the vehicle ahead to avoid collision if it suddenly brakes? |
| **2** | Does safe stopping distance depend on the speed of the vehicle? |
| **3** | How can motion be described not only in words, but also through numbers, equations, and graphs? |

## 4.1 — Motion in a Straight Line

When an object moves along a straight path, its motion is called **linear motion** or **motion in a straight line**.

Examples:

| Example | Why It Is Straight-Line Motion |
|---|---|
| A swimmer moving in a straight lane | Motion is along a nearly straight path |
| A ball falling vertically | Path is straight downward |
| A car on a straight highway | Motion is along a straight road segment |
| A train on a straight track | Track fixes a straight path |

Before describing motion, we must describe **position**.

---

## 4.1.1 — Describing Position

To describe the position of an object, we need a fixed point called a **reference point**.

> [!NOTE]
> A **reference point** is a fixed point with respect to which the position of an object is described.

The position of an object is described by:

| Part | Meaning |
|---|---|
| **Distance from reference point** | How far the object is |
| **Direction from reference point** | Which side or direction the object is |

If only distance is given, position may remain incomplete.

Example:

| Statement | Is It Complete? | Why |
|---|---|---|
| The athlete is 40 m from O | Not complete | Direction is missing |
| The athlete is 40 m to the right of O | Complete | Distance and direction are both given |

### Origin and Sign Convention

The reference point is often marked as the **origin**, written as **O**.

For straight-line motion, there are only two possible directions along the line. These are usually represented using signs:

| Direction from Origin | Sign |
|---|---|
| Right side / chosen positive direction | Positive `+` |
| Left side / opposite direction | Negative `−` |

> [!IMPORTANT]
> You may choose the positive direction for convenience, but once chosen, keep it the same throughout the problem.

### Rest and Motion

| Term | Meaning |
|---|---|
| **At rest** | Position does not change with time relative to the reference point |
| **In motion** | Position changes with time relative to the reference point |

> [!NOTE]
> Motion is always described **with respect to a reference point**. An object may be at rest with respect to one reference point and in motion with respect to another.

## 4.1.2 — Distance Travelled and Displacement

Distance and displacement are related, but they are not the same.

### Distance Travelled

> [!NOTE]
> **Distance travelled** is the total length of the path covered by an object.

Distance travelled:

| Feature | Description |
|---|---|
| Depends on path | Yes |
| Has direction? | No |
| Can be zero after motion? | No, unless the object did not move |
| SI unit | metre (m) |
| Quantity type | Scalar |

### Displacement

> [!NOTE]
> **Displacement** is the net change in position of an object between two instants of time.

Displacement:

| Feature | Description |
|---|---|
| Depends only on initial and final positions | Yes |
| Has direction? | Yes |
| Can be zero after motion? | Yes, if final position equals initial position |
| SI unit | metre (m) |
| Quantity type | Vector |

### Athlete Example

Suppose an athlete starts at O, runs to A, and then returns to B.

| Segment | Distance |
|---|---:|
| O to A | 100 m |
| A to B | 60 m |
| **Total distance travelled** | **160 m** |

If B is 40 m from O in the positive direction, then:

| Quantity | Value |
|---|---|
| Total distance travelled | 160 m |
| Displacement | 40 m in the positive direction |

So, distance travelled and displacement are different.

### Distance vs Displacement

| Feature | Distance Travelled | Displacement |
|---|---|---|
| Meaning | Total path length | Net change in position |
| Depends on route? | Yes | No |
| Direction needed? | No | Yes |
| Can be greater than displacement magnitude? | Yes | Not applicable |
| Can be zero after travelling? | No | Yes |
| Example | Full route walked | Straight-line change from start to end |

> [!TIP]
> Distance answers "How much path was covered?" Displacement answers "Where is the object now compared with where it started?"

### Scalars and Vectors

| Quantity Type | Meaning | Examples |
|---|---|---|
| **Scalar** | Has magnitude only | Distance, time, speed |
| **Vector** | Has magnitude and direction | Displacement, velocity, acceleration |

The numerical value with unit is called the **magnitude** of a physical quantity.

### When Are Distance and Displacement Equal?

For motion in a straight line:

| Situation | Result |
|---|---|
| Object moves in one direction without turning back | Distance travelled = magnitude of displacement |
| Object turns back | Distance travelled > magnitude of displacement |
| Object returns to starting point | Displacement = 0, but distance travelled is not zero |

### 🔬 Activity 4.1 — Analysing a Vertically Thrown Ball

**Aim:** To compare distance travelled and displacement for a ball moving up and down in a straight line.

#### Setup

A ball is thrown vertically upward from O, moves up to B, and falls back to O.

#### Observation Table

| Position | Total Distance Travelled from O | Displacement from O |
|---|---:|---|
| **O** | 0 cm | 0 cm |
| **A** | 40 cm | 40 cm upward |
| **B** | 80 cm | 80 cm upward |
| **C** | 120 cm | 40 cm upward |
| **O again** | 160 cm | 0 cm |

#### Inference

| Observation | Meaning |
|---|---|
| The ball moves in a straight vertical line | It is straight-line motion |
| Distance keeps increasing during the trip | Distance depends on total path |
| Displacement becomes zero when ball returns to O | Displacement depends only on final position relative to initial position |
| Displacement magnitude never exceeds distance | Magnitude of displacement is always less than or equal to total distance |

> [!NOTE]
> The magnitude of displacement can be **less than or equal to** the distance travelled, but it cannot be greater than the distance travelled.

## 4.1.3 — Average Speed and Average Velocity

To describe how fast or slow an object moves, we use speed and velocity.

### Average Speed

> [!NOTE]
> **Average speed** is the total distance travelled divided by the time interval.

$$
\text{Average speed}
=
\frac{\text{Total distance travelled}}{\text{Time interval}}
\tag{4.1}
$$

| Feature | Average Speed |
|---|---|
| Uses | Total distance travelled |
| Direction included? | No |
| Quantity type | Scalar |
| SI unit | m s⁻¹ or m/s |

### Uniform and Non-Uniform Motion

| Type of Motion | Meaning |
|---|---|
| **Uniform motion in a straight line** | Object travels equal distances in equal time intervals for all possible equal intervals |
| **Non-uniform motion in a straight line** | Object travels unequal distances in equal time intervals |

If distances covered in successive equal time intervals increase, speed is increasing. If they decrease, speed is decreasing.

### Average Velocity

Speed tells how fast an object moves, but not the direction of motion. Velocity includes direction.

> [!NOTE]
> **Average velocity** is displacement divided by the time interval.

$$
\text{Average velocity}
=
\frac{\text{Displacement}}{\text{Time interval}}
$$

If displacement is represented by `s` and time interval by `t`, then:

$$
v_{\text{av}}
=
\frac{s}{t}
\tag{4.2}
$$

| Feature | Average Velocity |
|---|---|
| Uses | Displacement |
| Direction included? | Yes |
| Quantity type | Vector |
| SI unit | m s⁻¹ or m/s |

### Average Speed vs Average Velocity

| Feature | Average Speed | Average Velocity |
|---|---|---|
| Formula uses | Total distance | Displacement |
| Direction | No direction | Direction required |
| Can be zero after motion? | No, if object actually moved | Yes, if displacement is zero |
| Example | Round trip has non-zero average speed | Round trip has zero average velocity |

### Rate of Change

> [!NOTE]
> The **rate of change** of a quantity is the change in that quantity divided by the time taken for the change.

Average velocity is the average rate of change of position with respect to time.

### 🇮🇳 India's Scientific Contributions — Speed in Ganitakaumudi

The idea that speed relates distance and time is ancient. A problem based on this idea appears in *Ganitakaumudi*, a mathematical text from the 14th century CE.

#### Example 4.1 — Two Postmen Walking Toward Each Other

Two postmen start walking toward each other from a distance of **210 yojanas**. One travels **9 yojanas per day** and the other travels **5 yojanas per day**.

| Quantity | Value |
|---|---:|
| Total separation | 210 yojanas |
| First postman speed | 9 yojanas/day |
| Second postman speed | 5 yojanas/day |
| Combined closing speed | 14 yojanas/day |

$$
\text{Time}
=
\frac{210}{9+5}
=
\frac{210}{14}
=
15\text{ days}
$$

So, they meet after **15 days**.

In 15 days:

| Postman | Distance Covered |
|---|---:|
| First postman | 135 yojanas |
| Second postman | 75 yojanas |

### 📐 Example 4.2 — Swimming Pool Round Trip

Sarang swims from one end of a pool to the other end and back. He takes **50 s**. The total distance travelled is **50 m**, but he ends where he started.

| Quantity | Value |
|---|---:|
| Total distance travelled | 50 m |
| Displacement | 0 m |
| Time interval | 50 s |

Average speed:

$$
\text{Average speed}
=
\frac{50\text{ m}}{50\text{ s}}
=
1\text{ m s}^{-1}
$$

Average velocity:

$$
\text{Average velocity}
=
\frac{0\text{ m}}{50\text{ s}}
=
0\text{ m s}^{-1}
$$

> [!TIP]
> A round trip can have non-zero average speed but zero average velocity.

## Instantaneous Velocity

Average velocity is calculated over a time interval. Velocity at a particular instant is called **instantaneous velocity**.

> [!NOTE]
> In this chapter, the word **velocity** is used to mean velocity at a particular instant, unless average velocity is specifically mentioned.

You will study instantaneous velocity in more detail in higher grades.

## 4.1.4 — Average Acceleration

When a vehicle suddenly starts, you feel a jolt. When it suddenly stops, you feel another jolt. These experiences are caused by a change in velocity.

> [!NOTE]
> **Average acceleration** is the change in velocity divided by the time interval.

$$
\text{Average acceleration}
=
\frac{\text{Change in velocity}}{\text{Time interval}}
\tag{4.3a}
$$

If initial velocity is `u`, final velocity is `v`, and the change occurs from time `t_1` to `t_2`, then:

$$
a
=
\frac{v-u}{t_2-t_1}
\tag{4.3c}
$$

If the initial time is taken as `0`, the time interval is `t`, so:

$$
a
=
\frac{v-u}{t}
$$

| Feature | Acceleration |
|---|---|
| Meaning | Rate of change of velocity |
| Direction included? | Yes |
| SI unit | m s⁻² or m/s² |
| Can be positive or negative? | Yes, depending on chosen direction |

### Direction of Acceleration

| Situation | Acceleration Direction |
|---|---|
| Speed increasing in the direction of motion | Same as velocity |
| Speed decreasing while moving forward | Opposite to velocity |
| Speed constant but direction changing | Acceleration exists because velocity changes |

> [!IMPORTANT]
> Acceleration does not mean only "speeding up". In physics, acceleration means **any change in velocity**, including change in direction.

---

### Unit Conversion

Many numerical problems use km h⁻¹ and m s⁻¹.

$$
1\text{ km h}^{-1}
=
\frac{1000\text{ m}}{3600\text{ s}}
=
\frac{5}{18}\text{ m s}^{-1}
$$

So:

$$
\text{Speed in m s}^{-1}
=
\text{Speed in km h}^{-1}
\times
\frac{5}{18}
$$

And:

$$
\text{Speed in km h}^{-1}
=
\text{Speed in m s}^{-1}
\times
\frac{18}{5}
$$

Quick conversions:

| Speed in km h⁻¹ | Speed in m s⁻¹ |
|---:|---:|
| 18 | 5 |
| 36 | 10 |
| 54 | 15 |
| 72 | 20 |
| 90 | 25 |
| 108 | 30 |

### 📐 Example 4.3 — Acceleration of a Bus

A bus moves on a straight highway with velocity **36 km h⁻¹**. The driver presses the accelerator for **10 s**, and the velocity increases to **54 km h⁻¹**. Later, the bus moving at **54 km h⁻¹** comes to rest in **5 s** after braking.

#### Step 1 — Convert Speeds

$$
36\text{ km h}^{-1}
=
36\times\frac{5}{18}
=
10\text{ m s}^{-1}
$$

$$
54\text{ km h}^{-1}
=
54\times\frac{5}{18}
=
15\text{ m s}^{-1}
$$

#### Case 1 — Accelerator Pressed

| Quantity | Value |
|---|---:|
| Initial velocity, `u` | 10 m s⁻¹ |
| Final velocity, `v` | 15 m s⁻¹ |
| Time, `t` | 10 s |

$$
a
=
\frac{v-u}{t}
=
\frac{15-10}{10}
=
0.5\text{ m s}^{-2}
$$

Acceleration is positive, so it acts in the direction of motion.

#### Case 2 — Brakes Pressed

| Quantity | Value |
|---|---:|
| Initial velocity, `u` | 15 m s⁻¹ |
| Final velocity, `v` | 0 m s⁻¹ |
| Time, `t` | 5 s |

$$
a
=
\frac{0-15}{5}
=
-3\text{ m s}^{-2}
$$

Acceleration is negative, so it acts opposite to the direction of motion.

> [!TIP]
> Negative acceleration does not always mean the object is moving backward. It means acceleration is opposite to the chosen positive direction.

### 📐 Example 4.4 — Falling Object and Acceleration Due to Gravity

An object is dropped from a height. Its velocity increases every second:

| Time | Velocity |
|---:|---:|
| 0 s | 0 m s⁻¹ |
| 1 s | 9.8 m s⁻¹ |
| 2 s | 19.6 m s⁻¹ |
| 3 s | 29.4 m s⁻¹ |
| 4 s | 39.2 m s⁻¹ |

Acceleration in each one-second interval:

$$
\frac{9.8-0}{1}
=
9.8\text{ m s}^{-2}
$$

$$
\frac{19.6-9.8}{1}
=
9.8\text{ m s}^{-2}
$$

$$
\frac{29.4-19.6}{1}
=
9.8\text{ m s}^{-2}
$$

$$
\frac{39.2-29.4}{1}
=
9.8\text{ m s}^{-2}
$$

The acceleration is constant and equal to **9.8 m s⁻²** downward.

> [!NOTE]
> The acceleration due to Earth's gravitational force is denoted by **g** and has magnitude approximately **9.8 m s⁻²** near Earth's surface.

## 4.2 — Graphical Representation of Motion

Graphs help us describe motion visually.

Instead of only saying "the car moved faster", a graph can show:

| Graph Shows | What We Can Learn |
|---|---|
| Position vs time | Where the object is at different times |
| Velocity vs time | How velocity changes with time |
| Slope | Rate of change |
| Area under velocity-time graph | Displacement |

In this chapter, graphs are considered for motion in a straight line in one direction. In this special case:

| Special Case | Result |
|---|---|
| Motion is in one direction only | Distance travelled = magnitude of displacement |
| Motion is in one direction only | Speed = magnitude of velocity |
| Position is zero at time zero | Position-time graph may match distance-time graph |

> [!IMPORTANT]
> A graph is not a route map. It does not show the physical road or path. It shows how a quantity changes with another quantity.

---

## 4.2.1 — Plotting a Graph

Suppose a vehicle moves along a straight road with the following data:

| Time (s) | 0 | 1 | 2 | 3 | 4 | 5 | 6 |
|---|---:|---:|---:|---:|---:|---:|---:|
| Position (m) | 0 | 20 | 40 | 60 | 80 | 100 | 120 |

### Steps for Plotting

| Step | Action |
|---|---|
| **1** | Draw two perpendicular lines on graph paper. |
| **2** | Mark their intersection as origin O. |
| **3** | Mark the horizontal line as X-axis and vertical line as Y-axis. |
| **4** | Put time on the X-axis. |
| **5** | Put position on the Y-axis. |
| **6** | Choose a suitable scale for each axis. |
| **7** | Mark values on both axes. |
| **8** | Plot each data point. |
| **9** | Join the points smoothly. |

Example scale:

| Axis | Scale |
|---|---|
| X-axis | 5 divisions = 1 s |
| Y-axis | 5 divisions = 20 m |

### Why Scale Matters

| Poor Scale Choice | Good Scale Choice |
|---|---|
| Graph too small or crowded | Graph uses available space well |
| Difficult to read values | Values are easy to locate |
| Shape may be misleading | Pattern becomes clear |

### 📐 Example 4.5 — Position-Time Graph for Speeding Up

For a vehicle starting from rest and speeding up:

| Time (s) | 0 | 2 | 4 | 6 | 8 | 10 | 12 |
|---|---:|---:|---:|---:|---:|---:|---:|
| Position (m) | 0 | 1 | 4 | 9 | 16 | 25 | 36 |

The graph is **curved**, not a straight line.

| Observation | Inference |
|---|---|
| Points do not lie on one straight line | Velocity is changing |
| Position increases more rapidly with time | Object is speeding up |
| Position-time graph is curved | Motion is accelerated |

## 4.2.2 — Position-Time Graphs

A position-time graph shows how position changes with time.

### Shape of Position-Time Graph

| Graph Shape | Meaning |
|---|---|
| Straight sloping line | Constant velocity |
| Curved line | Changing velocity / accelerated motion |
| Horizontal line | Object is at rest |
| Steeper line | Greater velocity magnitude |

### Slope of Position-Time Graph

From a position-time graph:

$$
\text{Average velocity}
=
\frac{\text{Change in position}}{\text{Change in time}}
$$

If position changes from `s_1` to `s_2` while time changes from `t_1` to `t_2`, then:

$$
v_{\text{av}}
=
\frac{s_2-s_1}{t_2-t_1}
$$

This ratio is the **slope** of the line joining the two points.

> [!NOTE]
> The slope of a position-time graph gives velocity.

### Example Slope Calculation

Suppose from the graph:

| Quantity | Value |
|---|---:|
| `s_1` | 40 m |
| `s_2` | 80 m |
| `t_1` | 2 s |
| `t_2` | 4 s |

$$
v
=
\frac{80-40}{4-2}
=
\frac{40}{2}
=
20\text{ m s}^{-1}
$$

### 📐 Example 4.6 — Horizontal Position-Time Graph

If a position-time graph is a straight line parallel to the time axis, the position is not changing.

| Graph Feature | Meaning |
|---|---|
| Position remains 40 m from origin | Object is at rest at 40 m |
| Slope is zero | Velocity is zero |

### 📐 Example 4.7 — Comparing Two Objects

If two objects A and B have straight position-time graphs and B's graph is steeper than A's graph:

| Object | Graph Slope | Velocity |
|---|---|---|
| A | Less steep | Lower velocity |
| B | More steep | Higher velocity |

> [!TIP]
> On a position-time graph, compare steepness to compare velocity.

## 4.2.3 — Velocity-Time Graphs

A velocity-time graph shows how velocity changes with time.

### Shape of Velocity-Time Graph

| Graph Shape | Meaning |
|---|---|
| Horizontal line | Constant velocity, zero acceleration |
| Straight line sloping upward | Velocity increasing with constant positive acceleration |
| Straight line sloping downward | Velocity decreasing with constant negative acceleration |
| Curved line | Acceleration is changing |

### Example Velocity Tables

#### Increasing Velocity

| Time (s) | 0 | 5 | 10 | 15 | 20 | 25 | 30 |
|---|---:|---:|---:|---:|---:|---:|---:|
| Velocity (m s⁻¹) | 0 | 2.5 | 5.0 | 7.5 | 10.0 | 12.5 | 15.0 |

Velocity increases by equal amounts in equal time intervals, so acceleration is constant and positive.

#### Decreasing Velocity

| Time (s) | 0 | 5 | 10 | 15 | 20 | 25 | 30 |
|---|---:|---:|---:|---:|---:|---:|---:|
| Velocity (m s⁻¹) | 15.0 | 12.5 | 10.0 | 7.5 | 5.0 | 2.5 | 0 |

Velocity decreases by equal amounts in equal time intervals, so acceleration is constant and negative.

### Slope of Velocity-Time Graph

From a velocity-time graph:

$$
\text{Acceleration}
=
\frac{\text{Change in velocity}}{\text{Change in time}}
$$

If velocity changes from `u` to `v` while time changes from `t_1` to `t_2`, then:

$$
a
=
\frac{v-u}{t_2-t_1}
$$

> [!NOTE]
> The slope of a velocity-time graph gives acceleration.

### Example Acceleration from Velocity-Time Graph

From `10 s` to `20 s`, suppose velocity changes from `5 m s⁻¹` to `10 m s⁻¹`.

$$
a
=
\frac{10-5}{20-10}
=
\frac{5}{10}
=
0.5\text{ m s}^{-2}
$$

If velocity decreases similarly, acceleration may be:

$$
a
=
-0.5\text{ m s}^{-2}
$$

### Area Under Velocity-Time Graph

For a velocity-time graph:

> [!NOTE]
> The area between the velocity-time graph and the time axis gives displacement.

For constant velocity:

$$
\text{Displacement}
=
\text{velocity}
\times
\text{time interval}
$$

Example:

$$
\text{Displacement}
=
20\text{ m s}^{-1}
\times
6\text{ s}
=
120\text{ m}
$$

For constant acceleration, the area can be split into a rectangle and a triangle:

$$
\text{Displacement}
=
\text{Area of rectangle}
+
\text{Area of triangle}
$$

Example from 10 s to 20 s:

$$
\text{Displacement}
=
(5\times10)
+
\left(\frac{1}{2}\times10\times5\right)
=
50+25
=
75\text{ m}
$$

### Graph Rules Summary

| Graph | Slope Gives | Area Gives |
|---|---|---|
| Position-time graph | Velocity | Not used here |
| Velocity-time graph | Acceleration | Displacement |

## 4.3 — Kinematic Equations for Motion in a Straight Line with Constant Acceleration

Kinematic equations connect displacement, time, initial velocity, final velocity, and acceleration.

They are valid only for motion in a straight line with **constant acceleration**.

### Symbols Used

| Symbol | Meaning | SI Unit |
|---|---|---|
| `u` | Initial velocity | m s⁻¹ |
| `v` | Final velocity | m s⁻¹ |
| `a` | Acceleration | m s⁻² |
| `t` | Time interval | s |
| `s` | Displacement | m |

### First Kinematic Equation

Start with acceleration:

$$
a
=
\frac{v-u}{t}
$$

Rearrange:

$$
at
=
v-u
$$

So:

$$
v
=
u+at
\tag{4.4a}
$$

This equation is used when final velocity is needed and `u`, `a`, and `t` are known.

### Second Kinematic Equation

From a velocity-time graph, displacement is area under the graph.

For constant acceleration:

$$
s
=
ut
+
\frac{1}{2}at^2
\tag{4.4b}
$$

This equation is used when displacement is needed and `u`, `a`, and `t` are known.

### Third Kinematic Equation

When time is not directly needed:

$$
v^2
=
u^2
+
2as
\tag{4.4c}
$$

This equation connects velocity, acceleration, and displacement.

### Complete Formula Set

| Equation | Best Used When |
|---|---|
| `v = u + at` | Need final velocity, and time is known |
| `s = ut + 1/2 at²` | Need displacement, and time is known |
| `v² = u² + 2as` | Need relation between speed and distance, and time is not needed |

> [!IMPORTANT]
> These equations are valid only when acceleration is constant. Do not use them blindly for changing acceleration.

### 📐 Example 4.8 — Braking Distance at Different Speeds

A car is moving on a highway. Brakes are applied, causing acceleration **−4 m s⁻²**. Find the distance travelled before stopping if the car was moving at:

| Case | Initial Speed |
|---|---:|
| **i** | 54 km h⁻¹ |
| **ii** | 108 km h⁻¹ |

Final velocity in both cases:

$$
v=0\text{ m s}^{-1}
$$

Use:

$$
v^2
=
u^2
+
2as
$$

Since `v = 0` and `a = −4 m s⁻²`:

$$
0
=
u^2
+
2(-4)s
$$

$$
0
=
u^2
-
8s
$$

So:

$$
s
=
\frac{u^2}{8}
$$

#### Case i — 54 km h⁻¹

$$
54\text{ km h}^{-1}
=
15\text{ m s}^{-1}
$$

$$
s
=
\frac{15^2}{8}
=
\frac{225}{8}
=
28.125\text{ m}
\approx
28.1\text{ m}
$$

#### Case ii — 108 km h⁻¹

$$
108\text{ km h}^{-1}
=
30\text{ m s}^{-1}
$$

$$
s
=
\frac{30^2}{8}
=
\frac{900}{8}
=
112.5\text{ m}
$$

### Key Inference

| Initial Speed | Stopping Distance |
|---:|---:|
| 54 km h⁻¹ | 28.1 m |
| 108 km h⁻¹ | 112.5 m |

Doubling speed made stopping distance four times larger in this example.

> [!IMPORTANT]
> Stopping distance depends strongly on initial speed. This is why maintaining safe distance from the vehicle ahead is essential.

## 🌉 Bridging Science and Society — Safe Stopping Distance

When brakes are applied, a moving vehicle does not stop instantly. It travels some distance before stopping.

Stopping distance depends on:

| Factor | Effect |
|---|---|
| Initial speed | Higher speed increases stopping distance |
| Road surface | Wet or slippery roads increase stopping distance |
| Braking capacity | Better brakes reduce stopping distance |
| Tyre condition | Worn tyres reduce grip |
| Vehicle mass and load | Heavier vehicles may need more distance |
| Driver reaction time | Vehicle keeps moving before brakes are applied |

### Thinking About Reaction Time

If a driver takes 0.5 s to react, the vehicle continues moving during those 0.5 s.

$$
\text{Reaction distance}
=
\text{speed}
\times
\text{reaction time}
$$

For a bus moving at 36 km h⁻¹:

$$
36\text{ km h}^{-1}
=
10\text{ m s}^{-1}
$$

$$
\text{Reaction distance}
=
10\times0.5
=
5\text{ m}
$$

The braking distance must be added to reaction distance to get total stopping distance.

### Vehicle-to-Vehicle Communication

Vehicle-to-vehicle communication technology allows vehicles to exchange signals and warn drivers of possible collisions. Such systems are being developed in many countries, including India.

## 4.4 — Motion in a Plane

So far, this chapter has mainly discussed motion in a straight line, also called **one-dimensional motion**.

Motion in a plane is **two-dimensional motion**.

Examples:

| Motion | Why It Is Motion in a Plane |
|---|---|
| A vehicle overtaking another | Position changes in two directions |
| A kicked ball | Path may be curved through a plane |
| A satellite in circular path | Motion involves changing direction |

Motion in space, such as a bird flying in the sky or an aircraft moving through air, is **three-dimensional motion**. You will study it in higher grades.

## 4.4.1 — Uniform Circular Motion

When an object moves along a circular path, its motion is called **circular motion**.

> [!NOTE]
> **Uniform circular motion** is motion along a circular path with constant speed.

### Distance and Displacement in Circular Motion

Suppose a child moves on a merry-go-round from A to B to C.

| Quantity | Meaning |
|---|---|
| Distance travelled | Length of curved path ABC |
| Displacement | Straight line from A to C |

After one complete revolution:

| Quantity | Value |
|---|---|
| Distance travelled | Circumference of circle |
| Displacement | 0 |

If radius is `R`, circumference is:

$$
\text{Circumference}
=
2\pi R
$$

If time for one revolution is `T`, then average speed is:

$$
v_{\text{av}}
=
\frac{2\pi R}{T}
\tag{4.5}
$$

Average velocity for one full revolution is:

$$
\text{Average velocity}
=
\frac{0}{T}
=
0
$$

### Constant Speed but Changing Velocity

In uniform circular motion:

| Quantity | Does It Change? | Why |
|---|---|---|
| Speed | No | Magnitude of velocity is constant |
| Direction of velocity | Yes | Object continuously changes direction |
| Velocity | Yes | Velocity includes direction |
| Acceleration | Yes | Velocity changes due to direction change |

> [!IMPORTANT]
> Uniform circular motion is accelerated motion because the **direction of velocity changes continuously**, even though speed remains constant.

### 🔬 Activity 4.5 — Ring and Marble Investigation

**Aim:** To observe the direction of motion of an object released from circular motion.

#### Materials

| Material | Purpose |
|---|---|
| Ring such as adhesive tape ring | Circular boundary |
| Marble | Moving object |
| Smooth surface | Allows marble to move easily |

#### Procedure

| Step | Action |
|---|---|
| **1** | Place the ring flat on a smooth surface. |
| **2** | Throw or flick the marble so that it moves along the inner boundary of the ring. |
| **3** | Predict what will happen if the ring is lifted while the marble is moving. |
| **4** | After one or two revolutions, lift the ring without disturbing the marble. |
| **5** | Observe the marble's path. |

#### Observation and Inference

| Observation | Inference |
|---|---|
| Marble does not continue in a circle after ring is lifted | Circular motion needed the ring's constraint |
| Marble moves in a straight line | It continues in the direction it had at the instant of release |
| Direction at a point on circle is along tangent | Velocity at that instant is tangential |

> [!NOTE]
> A **tangent** is a straight line that touches a circle at one and only one point.

### Polygon-to-Circle Thinking

Imagine an athlete running around different closed tracks:

| Track Shape | Direction Changes per Round |
|---|---:|
| Rectangle | 4 |
| Hexagon | 6 |
| Many-sided polygon | Many |
| Circle | Direction changes continuously |

This helps explain why circular motion has changing velocity direction at every point.

## ⚠️ Common Misconceptions

| Misconception | Correct Understanding |
|---|---|
| **Distance and displacement are always equal.** | They are equal only for straight-line motion in one direction without turning back. |
| **Displacement cannot be zero if distance is non-zero.** | A round trip has non-zero distance but zero displacement. |
| **Speed and velocity are the same.** | Speed has magnitude only; velocity has magnitude and direction. |
| **Negative acceleration always means moving backward.** | It means acceleration is opposite to the chosen positive direction. |
| **A fast object must have acceleration.** | A fast object has zero acceleration if velocity is constant. |
| **Uniform circular motion has no acceleration because speed is constant.** | It has acceleration because direction of velocity changes continuously. |
| **A graph is a map of the route.** | A graph shows relationship between quantities, not the actual route. |
| **The kinematic equations always work.** | They work only for constant acceleration. |
