# 1. Production and the Firm

## 1.1 Meaning of Production

Production is the process by which inputs are transformed into output.

Examples:

| Producer | Inputs Used | Output Produced |
| --- | --- | --- |
| Tailor | sewing machine, cloth, thread, labour | shirts |
| Farmer | land, labour, tractor, seed, fertiliser, water | wheat |
| Car manufacturer | factory land, machines, labour, steel, rubber, aluminium | cars |
| Rickshaw puller | rickshaw, labour | rides/transport service |
| Domestic helper | labour, tools | cleaning service |

Production can create goods or services.

## 1.2 Firm

A firm is a producing unit.

It acquires inputs, combines them, produces output, sells output, earns revenue, and tries to make profit.

Profit:

```text
Profit = Revenue - Cost
```

The chapter assumes that the firm's objective is to maximise profit.

## 1.3 Cost of Production

To acquire inputs, the firm has to pay for them.

This payment is called cost of production.

Examples:

- wages paid to workers
- rent for land/building
- price of raw materials
- cost of electricity
- payment for machinery

## 1.4 Simplifying Assumptions

The chapter uses a simple model.

Important assumptions:

1. Production is treated as instantaneous.
2. The terms production and supply are sometimes used closely.
3. The firm wants maximum profit.
4. The analysis begins with two inputs: labour and capital.

# 2. Production Function

## 2.1 Meaning

A production function shows the relation between inputs used and output produced.

It tells the maximum output that can be produced from different input combinations, given technology.

Simple form:

```text
q = f(L, K)
```

where:

`q = output`

`L = labour`

`K = capital`

## 2.2 Example

Suppose a farmer uses:

- land, `K`
- labour, `L`

One simple production function may be:

```text
q = K x L
```

If:

`K = 1`

`L = 2`

Then:

```text
q = 1 x 2 = 2
```

So output is 2 units.

## 2.3 Why It Shows Maximum Output

A production function assumes efficient use of inputs.

Efficient use means:

It is not possible to produce more output from the same input combination with the given technology.

So the production function gives maximum possible output, not careless or wasteful output.

## 2.4 Role of Technology

The production function is defined for a given technology.

If technology improves:

- same inputs can produce more output
- the production function changes
- maximum output for input combinations rises

## 2.5 Production Function Table

A production function can be shown in a table.

Rows may show labour.

Columns may show capital.

Each cell shows maximum output for that labour-capital combination.

Important idea:

If either labour or capital is zero, output may be zero if both inputs are necessary.

If both inputs are positive, output is positive.

Increasing an input generally increases output, though the rate of increase may change.

# 3. Isoquants

## 3.1 Meaning

An isoquant is a curve showing all combinations of two inputs that produce the same level of output.

It is similar in idea to an indifference curve, but for production.

Indifference curve:

Same satisfaction.

Isoquant:

Same output.

## 3.2 Example

Suppose 10 units of output can be produced by:

- 4 units of labour and 1 unit of capital
- 2 units of labour and 2 units of capital
- 1 unit of labour and 4 units of capital

All these combinations lie on the isoquant for output `q = 10`.

## 3.3 Why Isoquants Slope Downward

If marginal products are positive:

To keep output constant, using more labour requires using less capital.

If both labour and capital increase, output would rise and the firm would move to a higher isoquant.

Therefore, an isoquant is negatively sloped.

## 3.4 Higher Isoquant

A higher isoquant represents a higher level of output.

Example:

`q = 50` lies above `q = 10` because it represents more output.

# 4. Short Run and Long Run

## 4.1 Short Run

Short run is a period in which at least one factor of production cannot be varied.

The factor that cannot be changed is called the fixed factor.

The factor that can be changed is called the variable factor.

Example:

A firm may have fixed capital in the short run, but can hire more labour.

So:

`K = fixed`

`L = variable`

## 4.2 Long Run

Long run is a period in which all factors of production can be varied.

There is no fixed factor in the long run.

The firm can change:

- labour
- capital
- plant size
- machines
- land/building arrangements
- production scale

## 4.3 Important Point

Short run and long run are not defined by calendar time alone.

They are defined by whether inputs can be varied.

For one industry, long run may be a few months.

For another, it may be many years.

## 4.4 Comparison

| Basis | Short Run | Long Run |
| --- | --- | --- |
| Fixed factor | At least one fixed factor | No fixed factor |
| Variable factors | Some inputs variable | All inputs variable |
| Scale of production | Limited by fixed factors | Can be fully adjusted |
| Example | Factory size fixed, labour variable | Factory size and labour both variable |

# 5. Total Product, Average Product, and Marginal Product

## 5.1 Total Product

Total product is the total output produced by different quantities of a variable input when other inputs remain fixed.

It is also called:

- total return
- total physical product

Example:

If capital is fixed and labour changes, total product of labour shows output at each labour level.

## 5.2 Average Product

Average product is output per unit of variable input.

For labour:

```text
APL = TPL / L
```

where:

`APL = average product of labour`

`TPL = total product of labour`

`L = units of labour`

## 5.3 Marginal Product

Marginal product is the change in output due to one more unit of variable input, keeping other inputs fixed.

For labour:

```text
MPL = Delta TPL / Delta L
```

If labour increases by one unit:

```text
MPL = TP at L units - TP at (L - 1) units
```

## 5.4 Product Schedule

| Labour | Total Product | Marginal Product | Average Product |
| ---: | ---: | ---: | ---: |
| 0 | 0 | - | - |
| 1 | 10 | 10 | 10 |
| 2 | 24 | 14 | 12 |
| 3 | 40 | 16 | 13.33 |
| 4 | 50 | 10 | 12.5 |
| 5 | 56 | 6 | 11.2 |
| 6 | 57 | 1 | 9.5 |

## 5.5 Reading the Product Schedule

When labour rises from 1 to 2:

```text
MP = 24 - 10 = 14
```

When labour rises from 2 to 3:

```text
MP = 40 - 24 = 16
```

When labour rises from 3 to 4:

```text
MP = 50 - 40 = 10
```

MP first rises, then falls.

## 5.6 Relationship Between TP and MP

Total product is the sum of marginal products.

Example:

At labour 4:

```text
TP = 10 + 14 + 16 + 10 = 50
```

Important:

If MP is positive, TP increases.

If MP is zero, TP is maximum/constant.

If MP is negative, TP falls.

## 5.7 Relationship Between AP and MP

Average product is the average of marginal products up to that input level.

Rules:

| Situation | Result |
| --- | --- |
| MP > AP | AP rises |
| MP = AP | AP is maximum |
| MP < AP | AP falls |

Therefore:

**MP curve cuts AP curve from above at the maximum point of AP.**

# 6. Law of Variable Proportions

## 6.1 Meaning

The law of variable proportions states:

**When one input is increased while other inputs are fixed, marginal product of the variable input initially rises, but after a certain level of employment, it starts falling.**

It is also called the law of diminishing marginal product.

## 6.2 Why MP First Rises

Initially, the fixed factor may be underused.

Example:

A farmer has 4 hectares of land and only 1 worker.

One worker may not be enough to use the land efficiently.

Adding workers improves the combination of labour and land.

So each additional worker may add more output than the previous worker.

MP rises.

## 6.3 Why MP Eventually Falls

After a point, the fixed factor becomes crowded.

Example:

Too many workers on the same 4 hectares of land.

Each worker has less land and equipment to work with.

Additional workers add less extra output.

MP falls.

## 6.4 Factor Proportions

Factor proportion means the ratio in which inputs are combined.

When one factor is fixed and the other increases, factor proportions change.

At first, the changing proportion may improve productivity.

After a point, it worsens productivity.

This explains the law of variable proportions.

## 6.5 Shape of TP Curve

TP curve:

- slopes upward when MP is positive
- first rises at an increasing rate when MP rises
- later rises at a decreasing rate when MP falls but remains positive
- may fall if MP becomes negative

## 6.6 Shape of MP Curve

MP curve is usually inverse U-shaped.

It rises first and then falls.

## 6.7 Shape of AP Curve

AP curve is also usually inverse U-shaped.

It rises when MP is above AP.

It falls when MP is below AP.

MP cuts AP at AP's maximum.

# 7. Returns to Scale

## 7.1 Meaning

Returns to scale describe what happens to output when all inputs are increased in the same proportion.

This is a long-run concept because all inputs can vary only in the long run.

## 7.2 Constant Returns to Scale

If all inputs increase by a proportion and output increases by the same proportion, there are constant returns to scale.

Example:

Inputs double.

Output doubles.

This is CRS.

Mathematically:

```text
f(t x1, t x2) = t f(x1, x2)
```

## 7.3 Increasing Returns to Scale

If all inputs increase by a proportion and output increases by a larger proportion, there are increasing returns to scale.

Example:

Inputs double.

Output more than doubles.

This is IRS.

Mathematically:

```text
f(t x1, t x2) > t f(x1, x2)
```

## 7.4 Decreasing Returns to Scale

If all inputs increase by a proportion and output increases by a smaller proportion, there are decreasing returns to scale.

Example:

Inputs double.

Output less than doubles.

This is DRS.

Mathematically:

```text
f(t x1, t x2) < t f(x1, x2)
```

## 7.5 Comparison

| Inputs Increase | Output Increase | Return to Scale |
| --- | --- | --- |
| 2 times | 2 times | Constant returns |
| 2 times | More than 2 times | Increasing returns |
| 2 times | Less than 2 times | Decreasing returns |

## 7.6 Cobb-Douglas Production Function

A Cobb-Douglas production function may be written as:

```text
q = x1^alpha x2^beta
```

If both inputs are multiplied by `t`, new output becomes:

```text
q1 = t^(alpha + beta) q0
```

Rules:

| Condition | Returns to Scale |
| --- | --- |
| `alpha + beta = 1` | Constant returns to scale |
| `alpha + beta > 1` | Increasing returns to scale |
| `alpha + beta < 1` | Decreasing returns to scale |

# 8. Cost Function

## 8.1 Meaning

To produce output, the firm uses inputs.

Inputs have prices.

For a given output level, there may be many input combinations.

The firm chooses the least-cost input combination.

The cost function shows:

**the least cost of producing each output level, given input prices and technology.**

## 8.2 Why Least Cost Matters

A firm wants maximum profit.

Profit:

```text
Profit = Revenue - Cost
```

For any output level, lower cost means higher profit.

Therefore, firms choose least-cost production methods.

## 8.3 Example

Suppose 50 units of output can be produced by:

- 6 labour, 3 capital
- 4 labour, 4 capital
- 3 labour, 6 capital

The firm will compare costs of these combinations using input prices.

It will choose the combination with lowest cost.

# 9. Short-Run Costs

## 9.1 Total Fixed Cost

Total fixed cost, TFC, is the cost of fixed factors.

It does not change with output in the short run.

Even if output is zero, fixed cost may exist.

Examples:

- rent of factory
- salary of permanent manager
- fixed machinery cost

## 9.2 Total Variable Cost

Total variable cost, TVC, is the cost of variable factors.

It changes with output.

If output increases, TVC increases.

Examples:

- raw material cost
- wages of variable labour
- electricity used in production

At zero output, TVC is usually zero.

## 9.3 Total Cost

Total cost is the sum of total fixed cost and total variable cost.

Formula:

```text
TC = TFC + TVC
```

## 9.4 Cost Schedule

| Output | TFC | TVC | TC | AFC | AVC | SAC | SMC |
| ---: | ---: | ---: | ---: | ---: | ---: | ---: | ---: |
| 0 | 20 | 0 | 20 | - | - | - | - |
| 1 | 20 | 10 | 30 | 20 | 10 | 30 | 10 |
| 2 | 20 | 18 | 38 | 10 | 9 | 19 | 8 |
| 3 | 20 | 24 | 44 | 6.67 | 8 | 14.67 | 6 |
| 4 | 20 | 29 | 49 | 5 | 7.25 | 12.25 | 5 |
| 5 | 20 | 33 | 53 | 4 | 6.6 | 10.6 | 4 |
| 6 | 20 | 39 | 59 | 3.33 | 6.5 | 9.83 | 6 |
| 7 | 20 | 47 | 67 | 2.86 | 6.7 | 9.57 | 8 |
| 8 | 20 | 60 | 80 | 2.5 | 7.5 | 10 | 13 |
| 9 | 20 | 75 | 95 | 2.22 | 8.33 | 10.55 | 15 |
| 10 | 20 | 95 | 115 | 2 | 9.5 | 11.5 | 20 |

## 9.5 Average Fixed Cost

Average fixed cost is fixed cost per unit of output.

Formula:

```text
AFC = TFC / q
```

As output rises, AFC falls because the same fixed cost is spread over more units.

The AFC curve is downward sloping and approaches zero but does not touch zero.

It is a rectangular hyperbola.

## 9.6 Average Variable Cost

Average variable cost is variable cost per unit of output.

Formula:

```text
AVC = TVC / q
```

AVC is usually U-shaped.

It falls first, then rises.

## 9.7 Short-Run Average Cost

Short-run average cost is total cost per unit of output.

Formula:

```text
SAC = TC / q
```

Since:

```text
TC = TFC + TVC
```

Dividing by output:

```text
SAC = AFC + AVC
```

## 9.8 Short-Run Marginal Cost

Short-run marginal cost is the change in total cost due to one more unit of output.

Formula:

```text
SMC = Delta TC / Delta q
```

In the short run, fixed cost does not change.

Therefore:

```text
SMC = Delta TVC / Delta q
```

## 9.9 Example of SMC

At output 4:

`TC = 49`

At output 5:

`TC = 53`

Change in TC:

```text
53 - 49 = 4
```

Change in output:

```text
5 - 4 = 1
```

So:

```text
SMC = 4/1 = 4
```

## 9.10 Relationship Between MC and TVC

TVC is the sum of marginal costs up to that output level.

Example:

At output 5:

```text
TVC = SMC1 + SMC2 + SMC3 + SMC4 + SMC5
TVC = 10 + 8 + 6 + 5 + 4
TVC = 33
```

This matches the table.

## 9.11 Relationship Between AVC and SMC

AVC is the average of marginal costs up to that output level.

Rules:

| Situation | Result |
| --- | --- |
| SMC < AVC | AVC falls |
| SMC = AVC | AVC is minimum |
| SMC > AVC | AVC rises |

Therefore:

**SMC cuts AVC from below at the minimum point of AVC.**

## 9.12 Relationship Between SAC and SMC

Rules:

| Situation | Result |
| --- | --- |
| SMC < SAC | SAC falls |
| SMC = SAC | SAC is minimum |
| SMC > SAC | SAC rises |

Therefore:

**SMC cuts SAC from below at the minimum point of SAC.**

## 9.13 Why SMC Is U-Shaped

SMC is linked to marginal product.

Initially, MP of variable factor rises.

When MP rises, fewer extra inputs are needed for each extra unit of output.

So SMC falls.

After a point, MP falls.

When MP falls, more extra input is needed for each extra unit of output.

So SMC rises.

Therefore, SMC is U-shaped.

## 9.14 Shapes of Short-Run Cost Curves

| Curve | Shape | Reason |
| --- | --- | --- |
| TFC | Horizontal | Fixed cost does not change with output. |
| TVC | Upward sloping | More output requires more variable input. |
| TC | Upward sloping | TC = TFC + TVC. |
| AFC | Downward sloping rectangular hyperbola | Fixed cost spread over more output. |
| AVC | U-shaped | Falls then rises due to variable proportions. |
| SAC | U-shaped | Sum of AVC and AFC. |
| SMC | U-shaped | Falls then rises due to MP behaviour. |

## 9.15 SAC and AVC Difference

Since:

```text
SAC = AVC + AFC
```

SAC always lies above AVC.

The vertical distance between SAC and AVC is AFC.

As output rises, AFC falls.

Therefore, the distance between SAC and AVC becomes smaller.

## 9.16 Minimum of SAC vs Minimum of AVC

The minimum point of SAC lies to the right of the minimum point of AVC.

Reason:

Even after AVC starts rising, AFC is still falling. SAC may continue to fall until the rise in AVC becomes stronger than the fall in AFC.

# 10. Long-Run Costs

## 10.1 No Fixed Cost in the Long Run

In the long run, all inputs are variable.

Therefore, there are no fixed costs.

The firm can vary plant size, machines, labour, and other inputs.

## 10.2 Long-Run Average Cost

Long-run average cost is cost per unit of output when all inputs are variable.

Formula:

```text
LRAC = TC / q
```

## 10.3 Long-Run Marginal Cost

Long-run marginal cost is the change in total cost due to one additional unit of output in the long run.

Formula:

```text
LRMC = Delta TC / Delta q
```

If output changes from `q1 - 1` to `q1`, then:

```text
LRMC = TC at q1 - TC at (q1 - 1)
```

## 10.4 Shape of LRAC

LRAC is usually U-shaped.

Its shape is related to returns to scale.

| Region | Returns to Scale | LRAC Behaviour |
| --- | --- | --- |
| Initial region | Increasing returns to scale | LRAC falls |
| Middle region | Constant returns to scale | LRAC is constant/minimum |
| Later region | Decreasing returns to scale | LRAC rises |

## 10.5 Why IRS Makes LRAC Fall

Under increasing returns to scale:

Inputs rise by a certain proportion, but output rises by a larger proportion.

Cost rises less than output.

So cost per unit falls.

Therefore, LRAC falls.

## 10.6 Why DRS Makes LRAC Rise

Under decreasing returns to scale:

Inputs must rise by a larger proportion to get a given proportional increase in output.

Cost rises more than output.

So cost per unit rises.

Therefore, LRAC rises.

## 10.7 Why CRS Makes LRAC Constant

Under constant returns to scale:

Inputs and output rise in the same proportion.

Cost and output rise proportionately.

So average cost remains constant.

## 10.8 Relationship Between LRMC and LRAC

Rules:

| Situation | Result |
| --- | --- |
| LRMC < LRAC | LRAC falls |
| LRMC = LRAC | LRAC is minimum |
| LRMC > LRAC | LRAC rises |

Therefore:

**LRMC cuts LRAC from below at the minimum point of LRAC.**

# 11. Product-Cost Connection

## 11.1 MP and MC Move Oppositely

When marginal product rises, marginal cost falls.

When marginal product falls, marginal cost rises.

Reason:

If each extra worker adds more output, the cost of producing one extra unit falls.

If each extra worker adds less output, the cost of producing one extra unit rises.

## 11.2 AP and AVC Move Oppositely

Average product and average variable cost are inversely related when input price is constant.

If average product rises, fewer variable inputs are needed per unit of output, so AVC falls.

If average product falls, more variable inputs are needed per unit of output, so AVC rises.

## 11.3 Useful Memory Table

| Production Side | Cost Side |
| --- | --- |
| MP rises | MC falls |
| MP falls | MC rises |
| AP rises | AVC falls |
| AP falls | AVC rises |
| MP cuts AP at AP maximum | MC cuts AVC at AVC minimum |

# 12. Calculation Methods

## 12.1 From TP to AP and MP

Given total product:

```text
AP = TP / L
MP = change in TP / change in L
```

Example:

If:

`L = 3`

`TP = 40`

Then:

```text
AP = 40/3 = 13.33
```

If previous TP at `L = 2` is 24:

```text
MP = 40 - 24 = 16
```

## 12.2 From AP to TP and MP

Given AP:

```text
TP = AP x L
```

Then:

```text
MP = change in TP
```

## 12.3 From MP to TP and AP

Given MP and `TP = 0` at zero labour:

```text
TP at L = sum of all MP up to L
```

Then:

```text
AP = TP / L
```

## 12.4 From TC to Cost Schedules

If TC is given and TFC is known:

```text
TVC = TC - TFC
AFC = TFC / q
AVC = TVC / q
SAC = TC / q
SMC = change in TC / change in q
```

## 12.5 If AFC at One Output Is Given

Suppose AFC at 4 units is Rs 5.

Then:

```text
TFC = AFC x output
TFC = 5 x 4
TFC = Rs 20
```

Once TFC is known, all other cost values can be calculated.

## 12.6 From SMC to TVC, TC, AVC, SAC

If SMC schedule and TFC are given:

```text
TVC at q = sum of SMC up to q
TC = TFC + TVC
AVC = TVC / q
SAC = TC / q
```

At zero output:

```text
TVC = 0
TC = TFC
```

## 13.3 Law of Diminishing Marginal Product

Write:

The law of diminishing marginal product states that when one input is increased while other inputs are fixed, marginal product of the variable input initially rises but after a certain level of employment begins to fall. This happens because factor proportions first become more suitable, but later the fixed factor becomes crowded.

## 13.4 Returns to Scale

Write:

Returns to scale describe the change in output when all inputs are increased in the same proportion. If output increases in the same proportion, there are constant returns to scale. If output increases by a larger proportion, there are increasing returns to scale. If output increases by a smaller proportion, there are decreasing returns to scale.

## 13.5 Cost Function

Write:

A cost function gives the minimum cost of producing each level of output, given input prices and technology. Since a given output can often be produced by different input combinations, a profit-maximising firm chooses the least-cost combination.

## 13.6 Fixed, Variable, and Total Cost

Write:

Total fixed cost is the cost of fixed factors and does not change with output in the short run. Total variable cost is the cost of variable factors and changes with output. Total cost is the sum of total fixed cost and total variable cost, so `TC = TFC + TVC`.

## 13.7 Why AFC Falls

Write:

Average fixed cost is `TFC/q`. Since total fixed cost remains constant while output increases, fixed cost is spread over more units. Therefore AFC continuously falls as output rises. The AFC curve is a rectangular hyperbola.

## 13.8 Why SMC Cuts AVC at AVC Minimum

Write:

AVC is the average of marginal costs up to that output level. When SMC is less than AVC, AVC falls. When SMC is greater than AVC, AVC rises. Therefore, when SMC equals AVC, AVC is at its minimum. Hence, SMC cuts AVC from below at the minimum point of AVC.

## 13.9 Why SMC Cuts SAC at SAC Minimum

Write:

SAC is total cost per unit. When SMC is less than SAC, the average cost falls. When SMC is greater than SAC, the average cost rises. Therefore, when SMC equals SAC, SAC is minimum. Hence, SMC cuts SAC from below at the minimum point of SAC.

## 13.10 Long-Run Cost Curves

Write:

In the long run all inputs are variable and there are no fixed costs. LRAC is the cost per unit of output and LRMC is the extra cost of one more unit of output. LRAC is usually U-shaped: it falls under increasing returns to scale, is constant under constant returns to scale, and rises under decreasing returns to scale. LRMC cuts LRAC from below at the minimum point of LRAC.

# 14. Common Mistakes

| Mistake | Correction |
| --- | --- |
| Saying production function shows actual output even if inputs are wasted | It shows maximum output with efficient input use. |
| Defining short run as one year | Short run means at least one input is fixed; time length differs by industry. |
| Confusing total product and marginal product | TP is total output; MP is extra output from one more input unit. |
| Forgetting MP at zero input is undefined | Marginal product requires a change from a previous input level. |
| Thinking MP always falls | MP may initially rise, then fall. |
| Confusing law of variable proportions with returns to scale | Variable proportions changes one input with others fixed; returns to scale changes all inputs. |
| Saying fixed cost exists in long run | In long run all factors are variable, so there is no fixed cost. |
| Calculating SMC from AFC | SMC is change in TC or TVC, not change in AFC. |
| Adding AFC and SMC to get SAC | SAC = AFC + AVC, not AFC + SMC. |
| Saying MC cuts AC from above | MC cuts AVC/SAC/LRAC from below at their minimum points. |

# 16. Final One-Page Summary

Production is the transformation of inputs into output. A firm uses inputs such as labour and capital to produce goods or services. The production function shows the maximum output possible from different input combinations, given technology. It is written as `q = f(L,K)`.

In the short run, at least one factor is fixed and another is variable. In the long run, all factors are variable. An isoquant shows input combinations that produce the same output. Total product is total output from a variable input, average product is output per unit of variable input, and marginal product is the extra output from one more unit of input.

The law of variable proportions states that when one input increases while other inputs remain fixed, marginal product first rises and then falls. This happens because factor proportions first improve and later become unsuitable as the fixed factor gets crowded. MP cuts AP from above at the maximum of AP.

Returns to scale describe output changes when all inputs increase proportionately. Constant returns mean output rises in the same proportion. Increasing returns mean output rises more than proportionately. Decreasing returns mean output rises less than proportionately.

The cost function shows the minimum cost of producing each output level. In the short run, total cost equals total fixed cost plus total variable cost: `TC = TFC + TVC`. Average costs are calculated by dividing by output: `AFC = TFC/q`, `AVC = TVC/q`, and `SAC = TC/q = AFC + AVC`. Marginal cost is the change in total cost from producing one more unit: `SMC = Delta TC/Delta q`.

TFC is horizontal, AFC falls continuously, and TVC and TC rise with output. SMC, AVC, and SAC are U-shaped. SMC cuts AVC and SAC from below at their minimum points. In the long run, all costs are variable, so there is no fixed cost. LRAC is usually U-shaped due to increasing, constant, and decreasing returns to scale. LRMC cuts LRAC from below at the minimum point of LRAC.
