## Chapter Purpose

This chapter studies **hydrocarbons**, the organic compounds made only of carbon and hydrogen.

Hydrocarbons look simple at first because they contain only two elements, but they form a very large and important part of organic chemistry. Petrol, diesel, kerosene, LPG, CNG, natural gas, polythene, polypropylene, benzene derivatives, dyes, drugs, solvents, and many industrial raw materials are connected with hydrocarbons.

The chapter has one central purpose:

**Learn how the type of carbon-carbon bond controls structure, isomerism, preparation, reactions, and uses of hydrocarbons.**

If the bond is only single, the compound behaves like an alkane.

If there is a double bond, the compound behaves like an alkene.

If there is a triple bond, the compound behaves like an alkyne.

If the compound has a stable delocalised cyclic pi system like benzene, it behaves like an aromatic hydrocarbon.

This chapter is not just a list of reactions. It is a map of organic reactivity.

## Core Classification

Hydrocarbons are classified in two connected ways:

1. By carbon skeleton.
2. By type of carbon-carbon bond.

| Basis | Class | Meaning | Example |
| --- | --- | --- | --- |
| Open chain | Aliphatic hydrocarbons | Carbon atoms form open chains, straight or branched. | Butane, propene, propyne |
| Closed chain | Alicyclic hydrocarbons | Carbon atoms form non-aromatic rings. | Cyclohexane |
| Aromatic ring | Aromatic hydrocarbons | Stable cyclic conjugated systems, often benzene-like. | Benzene, toluene |
| Single bonds only | Saturated hydrocarbons | No C=C or C#C bond. | Alkanes |
| Multiple bonds | Unsaturated hydrocarbons | Contain C=C, C#C, or both. | Alkenes, alkynes |

# 1. Hydrocarbons in Daily Life

## 1.1 Meaning

The word **hydrocarbon** means:

`hydrogen + carbon`

A hydrocarbon contains only:

- carbon atoms
- hydrogen atoms

No oxygen, nitrogen, halogen, sulphur, or other element is present in a pure hydrocarbon.

Examples:

| Compound | Formula | Class |
| --- | --- | --- |
| Methane | `CH4` | Alkane |
| Ethane | `C2H6` | Alkane |
| Ethene | `C2H4` | Alkene |
| Ethyne | `C2H2` | Alkyne |
| Benzene | `C6H6` | Aromatic hydrocarbon |
| Toluene | `C6H5CH3` | Aromatic hydrocarbon |

## 1.2 Why Hydrocarbons Matter

Hydrocarbons are important because they are:

| Use | Explanation |
| --- | --- |
| Fuels | LPG, CNG, petrol, diesel, kerosene, coal gas, natural gas. |
| Energy sources | Combustion of hydrocarbons releases heat. |
| Polymer raw materials | Ethene gives polythene; propene gives polypropylene. |
| Industrial starting materials | Used to make dyes, drugs, solvents, plastics, and synthetic fibres. |
| Solvents | Higher hydrocarbons dissolve many non-polar substances. |
| Petrochemicals | Petroleum-derived hydrocarbons act as feedstock for chemical industries. |

# 2. Classification of Hydrocarbons

## 2.1 Saturated Hydrocarbons

Saturated hydrocarbons contain only single bonds between carbon atoms.

They have:

- C-C single bonds
- C-H single bonds
- only sigma bonds
- no carbon-carbon pi bond

Main example:

**Alkanes**

Example:

`CH3-CH3` is ethane.

It contains one C-C single bond and six C-H bonds.

## 2.2 Unsaturated Hydrocarbons

Unsaturated hydrocarbons contain at least one multiple bond.

They may contain:

- C=C double bond
- C#C triple bond
- both types in some compounds

Main examples:

| Class | Multiple Bond | Example |
| --- | --- | --- |
| Alkene | C=C | `CH2=CH2` |
| Alkyne | C#C | `HC#CH` |

The multiple bond contains a pi bond. The pi bond is weaker and more exposed than a sigma bond, so unsaturated hydrocarbons easily undergo addition reactions.

## 2.3 Aromatic Hydrocarbons

Aromatic hydrocarbons are cyclic, highly stable hydrocarbons with delocalised pi electrons.

The most important example is benzene, `C6H6`.

Benzene contains a ring of six carbon atoms, but it does not behave like an ordinary alkene. It usually undergoes substitution reactions instead of addition reactions.

## 2.4 Quick Identification Table

| Formula/Structure Clue | Likely Class |
| --- | --- |
| Only C-C single bonds in open chain | Alkane |
| Contains C=C | Alkene |
| Contains C#C | Alkyne |
| Contains benzene ring | Aromatic hydrocarbon |
| Ring with only single bonds and no aromaticity | Cycloalkane |

## 2.5 Decision Tree

Use this order:

1. Does it contain only C and H?
   - If no, it is not a hydrocarbon.
   - If yes, continue.
2. Does it contain benzene-like aromatic ring?
   - If yes, it is aromatic.
3. Does it contain C#C?
   - If yes, it is an alkyne.
4. Does it contain C=C?
   - If yes, it is an alkene.
5. Does it contain only C-C single bonds?
   - If yes, it is an alkane or cycloalkane.

# 3. Alkanes

## 3.1 Meaning

Alkanes are saturated open-chain hydrocarbons.

They contain:

- carbon-carbon single bonds
- carbon-hydrogen single bonds
- only sigma bonds
- no pi bonds

General formula:

`CnH2n+2`

Examples:

| n | Formula | Name |
| --- | --- | --- |
| 1 | `CH4` | Methane |
| 2 | `C2H6` | Ethane |
| 3 | `C3H8` | Propane |
| 4 | `C4H10` | Butane |
| 5 | `C5H12` | Pentane |

## 3.2 Why Alkanes Were Called Paraffins

Alkanes were earlier called **paraffins**.

The word comes from the idea of "little affinity".

This means alkanes show little tendency to react under ordinary conditions.

Reason:

- C-C sigma bonds are strong.
- C-H sigma bonds are strong.
- The molecule is nearly non-polar.
- No pi bond is present for electrophiles to attack.

## 3.3 Structure of Methane and Alkanes

Methane has formula `CH4`.

Carbon in methane is `sp3` hybridised.

Shape:

- tetrahedral
- bond angle about `109.5°`

In higher alkanes:

- carbon atoms are also mostly `sp3` hybridised
- C-C bond length is about `154 pm`
- C-H bond length is about `112 pm`
- each C-C and C-H bond is a sigma bond

## 3.4 Chain Growth in Alkanes

Alkanes can be imagined as methane molecules extended by replacing one hydrogen with `-CH3`.

| Step | Formula | Name |
| --- | --- | --- |
| Methane | `CH4` | first alkane |
| Replace H by `-CH3` | `CH3-CH3` | ethane |
| Extend again | `CH3-CH2-CH3` | propane |
| Extend again | `CH3-CH2-CH2-CH3` | butane |

Each next member differs from the previous member by `-CH2-`.

This is why alkanes form a homologous series.

## 3.5 Isomerism in Alkanes

The first three alkanes have only one possible carbon skeleton:

- methane
- ethane
- propane

From butane onward, different carbon skeletons are possible.

### Butane: `C4H10`

Butane has two chain isomers:

| Structure | Name |
| --- | --- |
| `CH3-CH2-CH2-CH3` | Butane or n-butane |
| `(CH3)3CH` | 2-Methylpropane or isobutane |

### Pentane: `C5H12`

Pentane has three chain isomers:

| Structure | Name |
| --- | --- |
| `CH3-CH2-CH2-CH2-CH3` | Pentane |
| `CH3-CH(CH3)-CH2-CH3` | 2-Methylbutane |
| `C(CH3)4` | 2,2-Dimethylpropane |

### Important Pattern

As the number of carbon atoms increases, the number of possible isomers increases rapidly.

Examples:

| Formula | Number of Chain Isomers |
| --- | --- |
| `C4H10` | 2 |
| `C5H12` | 3 |
| `C6H14` | 5 |
| `C7H16` | 9 |
| `C10H22` | 75 |

## 3.6 Primary, Secondary, Tertiary, and Quaternary Carbon

Carbon atoms in alkanes are classified by the number of carbon atoms directly attached to them.

| Type | Attached to How Many Carbon Atoms? | Example Meaning |
| --- | --- | --- |
| Primary carbon, `1°` | 0 or 1 carbon atom | terminal carbon in a chain |
| Secondary carbon, `2°` | 2 carbon atoms | middle carbon in a straight chain |
| Tertiary carbon, `3°` | 3 carbon atoms | branching carbon attached to three carbons |
| Quaternary carbon, `4°` | 4 carbon atoms | carbon attached to four carbons |

Example:

`CH3-CH(CH3)-CH3`

The middle carbon is tertiary because it is attached to three `CH3` groups.

Each `CH3` carbon is primary.

## 3.7 Alkyl Groups

An alkyl group is formed by removing one hydrogen atom from an alkane.

General formula:

`CnH2n+1-`

Examples:

| Alkane | Alkyl Group | Name |
| --- | --- | --- |
| Methane, `CH4` | `CH3-` | Methyl |
| Ethane, `C2H6` | `C2H5-` | Ethyl |
| Propane, `C3H8` | `C3H7-` | Propyl |
| Butane, `C4H10` | `C4H9-` | Butyl |

Alkyl groups appear as substituents in IUPAC names.

## 3.8 Naming Alkanes: Learner Method

To name a substituted alkane:

1. Choose the longest continuous carbon chain.
2. Name the parent chain.
3. Number the chain from the end giving the lowest locants to substituents.
4. Identify substituents.
5. Arrange substituents alphabetically.
6. Use prefixes such as di, tri, tetra for repeated substituents.
7. Write locants clearly.

Example:

`CH3-CH(CH3)-CH2-CH3`

Longest chain = 4 carbons.

Parent = butane.

Substituent = methyl at carbon 2.

Name = **2-methylbutane**.

## 3.9 Common Naming Mistake

Incorrect name:

`2-ethylpentane`

Why incorrect?

Because the longest chain is not pentane. If the ethyl branch is included in the main chain, the longest chain has six carbons.

Correct name:

**3-methylhexane**

Exam rule:

**Never choose the chain that looks horizontal automatically. Choose the longest continuous chain.**

# 4. Preparation of Alkanes

## 4.1 From Unsaturated Hydrocarbons: Hydrogenation

Alkenes and alkynes add hydrogen in the presence of finely divided catalysts such as platinum, palladium, or nickel.

This converts multiple bonds into single bonds.

General idea:

`alkene/alkyne + H2 -> alkane`

Examples:

```text
CH2=CH2 + H2 --Pt/Pd/Ni--> CH3-CH3
Ethene                         Ethane
```

```text
CH3-CH=CH2 + H2 --Pt/Pd/Ni--> CH3-CH2-CH3
Propene                         Propane
```

```text
CH3-C#CH + 2H2 --Pt/Pd/Ni--> CH3-CH2-CH3
Propyne                         Propane
```

Why catalyst is needed:

- catalyst adsorbs hydrogen on its surface
- H-H bond becomes activated
- hydrogen atoms add across multiple bond

## 4.2 From Alkyl Halides: Reduction

Alkyl halides, except fluorides, give alkanes on reduction with zinc and dilute hydrochloric acid.

General idea:

```text
R-X + 2[H] -> R-H + HX
```

Example:

```text
CH3Cl + 2[H] --Zn/HCl--> CH4 + HCl
Chloromethane              Methane
```

## 4.3 From Alkyl Halides: Wurtz Reaction

Alkyl halides react with sodium metal in dry ether to form higher alkanes.

General equation:

```text
2 R-X + 2 Na --dry ether--> R-R + 2 NaX
```

Example:

```text
2 CH3Br + 2 Na --dry ether--> CH3-CH3 + 2 NaBr
```

Important points:

| Point | Meaning |
| --- | --- |
| Dry ether is used | Moisture interferes with sodium. |
| Best for symmetrical alkanes | Same alkyl halide gives one major alkane. |
| Not preferred for odd-carbon alkanes | Two different alkyl halides give mixture of products. |
| Useful for even-carbon alkanes | Example: methyl bromide gives ethane; ethyl bromide gives butane. |

### Why Wurtz Reaction Is Not Good for Odd Alkanes

To prepare propane by Wurtz reaction, one might try:

`CH3Br + C2H5Br + 2Na`

But this gives a mixture:

- ethane from `CH3Br + CH3Br`
- propane from `CH3Br + C2H5Br`
- butane from `C2H5Br + C2H5Br`

Because of this mixture, the desired odd-carbon alkane is difficult to isolate.

## 4.4 From Carboxylic Acids: Decarboxylation

Sodium salts of carboxylic acids, when heated with soda lime, give alkanes containing one carbon atom fewer than the acid salt.

Soda lime:

`NaOH + CaO`

General equation:

```text
RCOONa + NaOH --CaO, heat--> RH + Na2CO3
```

Example:

```text
CH3COONa + NaOH --CaO, heat--> CH4 + Na2CO3
Sodium ethanoate               Methane
```

To prepare propane:

Use sodium butanoate.

```text
CH3CH2CH2COONa + NaOH --CaO, heat--> CH3CH2CH3 + Na2CO3
```

Reason:

The carboxyl carbon is lost as carbonate/carbon dioxide equivalent, so the product has one carbon less.

## 4.5 Kolbe's Electrolytic Method

An aqueous solution of sodium or potassium salt of a carboxylic acid gives an alkane at the anode on electrolysis.

Example:

```text
2 CH3COONa + 2 H2O --electrolysis--> CH3-CH3 + 2 CO2 + H2 + 2 NaOH
```

Important features:

| Feature | Explanation |
| --- | --- |
| Occurs at anode | Carboxylate ions lose electrons. |
| Free radicals form | Alkyl radicals combine to give alkane. |
| Even-carbon alkanes | Two identical radicals couple. |
| Methane not prepared | Coupling of radicals gives at least two carbon atoms for acetate-type cases. |

# 5. Physical Properties of Alkanes

## 5.1 Polarity

Alkanes are almost non-polar.

Reason:

- C-C bond is non-polar.
- C-H bond is only weakly polar.
- overall molecule has no strong dipole.

## 5.2 Solubility

Alkanes are:

- insoluble in water
- soluble in non-polar organic solvents

Reason:

Water is polar. Alkanes are non-polar.

Rule:

**Like dissolves like.**

This also explains why petrol and lower petroleum fractions can remove grease stains. Grease is a mixture of higher non-polar hydrocarbons, so it dissolves better in non-polar solvents than in water.

## 5.3 Physical State

At room temperature:

| Carbon Atoms | Physical State |
| --- | --- |
| `C1` to `C4` | Gases |
| `C5` to `C17` | Liquids |
| `C18` and above | Solids |

## 5.4 Boiling Point Trend

Boiling point generally increases with molecular mass.

Reason:

- larger molecules have larger surface area
- larger surface area gives stronger van der Waals forces
- more energy is needed to separate molecules

## 5.5 Effect of Branching on Boiling Point

For isomeric alkanes:

**More branching means lower boiling point.**

Reason:

- branching makes the molecule more compact
- compact molecules have smaller surface contact
- weaker van der Waals attraction
- lower boiling point

Example:

| Isomer of `C5H12` | Boiling Point Trend |
| --- | --- |
| Pentane | Highest |
| 2-Methylbutane | Lower |
| 2,2-Dimethylpropane | Lowest |

Answer frame:

Branching lowers boiling point because it reduces surface area and weakens intermolecular van der Waals forces.

# 6. Chemical Properties of Alkanes

Alkanes are generally unreactive, but under suitable conditions they show important reactions.

Main reactions:

1. Substitution reactions.
2. Combustion.
3. Controlled oxidation.
4. Isomerisation.
5. Aromatisation.
6. Reaction with steam.
7. Pyrolysis/cracking.

---

## 6.1 Substitution Reactions

In substitution, one atom or group in a molecule is replaced by another atom or group.

For alkanes:

Hydrogen atoms can be replaced by halogens.

Example:

```text
CH4 + Cl2 --hv--> CH3Cl + HCl
```

Further chlorination may give:

```text
CH3Cl + Cl2 --hv--> CH2Cl2 + HCl
CH2Cl2 + Cl2 --hv--> CHCl3 + HCl
CHCl3 + Cl2 --hv--> CCl4 + HCl
```

Products:

| Formula | Name |
| --- | --- |
| `CH3Cl` | Chloromethane |
| `CH2Cl2` | Dichloromethane |
| `CHCl3` | Trichloromethane/chloroform |
| `CCl4` | Tetrachloromethane/carbon tetrachloride |

## 6.2 Conditions for Halogenation

Halogenation occurs:

- at high temperature
- in diffused sunlight
- in ultraviolet light

Rate of halogen reaction:

`F2 > Cl2 > Br2 > I2`

Important notes:

| Halogen | Behaviour |
| --- | --- |
| Fluorine | Too violent to control. |
| Chlorine | Commonly used in examples. |
| Bromine | Slower and more selective. |
| Iodine | Very slow and reversible; needs oxidising agent. |

Rate of replacement of hydrogen:

`3° H > 2° H > 1° H`

Reason:

More substituted free radicals are more stable.

## 6.3 Free Radical Chain Mechanism of Chlorination

Chlorination of methane proceeds through a free radical chain mechanism.

There are three stages:

1. Initiation.
2. Propagation.
3. Termination.

### Stage 1: Initiation

Light breaks the chlorine molecule homolytically.

```text
Cl-Cl --hv--> Cl. + Cl.
```

This forms chlorine free radicals.

### Stage 2: Propagation

Step 1:

```text
Cl. + CH4 -> HCl + CH3.
```

Chlorine radical abstracts hydrogen from methane.

Step 2:

```text
CH3. + Cl2 -> CH3Cl + Cl.
```

Methyl radical attacks chlorine molecule and produces chloromethane plus a new chlorine radical.

The new chlorine radical continues the chain.

### Stage 3: Termination

Free radicals combine and the chain stops.

Examples:

```text
Cl. + Cl. -> Cl2
CH3. + CH3. -> CH3-CH3
CH3. + Cl. -> CH3Cl
```

### Why Ethane Forms During Methane Chlorination

Ethane forms because two methyl radicals may combine:

```text
CH3. + CH3. -> CH3-CH3
```

This is a termination step.

## 6.4 Combustion of Alkanes

Alkanes burn in oxygen to give carbon dioxide, water, and heat.

Example:

```text
CH4 + 2O2 -> CO2 + 2H2O
```

Combustion is highly exothermic.

This is why hydrocarbons are useful fuels.

General combustion equation for hydrocarbon:

```text
CxHy + (x + y/4)O2 -> xCO2 + (y/2)H2O
```

Example for butane:

```text
2C4H10 + 13O2 -> 8CO2 + 10H2O
```

## 6.5 Incomplete Combustion

If oxygen is insufficient, incomplete combustion can give carbon monoxide or carbon.

Example:

```text
CH4 + O2 -> C + 2H2O
```

Important:

Incomplete combustion causes pollution and can produce poisonous carbon monoxide.

## 6.6 Controlled Oxidation

Alkanes resist oxidation under ordinary conditions, but controlled oxidation under specific conditions gives useful products.

Examples:

```text
2CH4 + O2 --Cu, 523 K, 100 atm--> 2CH3OH
Methane                              Methanol
```

```text
CH4 + O2 --Mo2O3, heat--> HCHO + H2O
Methane                         Methanal
```

```text
2CH3CH3 + 3O2 --manganese acetate, heat--> 2CH3COOH + 2H2O
Ethane                                      Ethanoic acid
```

Alkanes with tertiary hydrogen can be oxidised to tertiary alcohols.

Example:

```text
(CH3)3CH --KMnO4--> (CH3)3COH
2-Methylpropane      2-Methylpropan-2-ol
```

## 6.7 Isomerisation

Straight-chain alkanes can convert into branched-chain alkanes.

Conditions:

- anhydrous aluminium chloride
- hydrogen chloride gas
- heating

Example idea:

```text
n-Hexane --AlCl3/HCl--> 2-methylpentane + 3-methylpentane
```

Importance:

Branched alkanes often have better fuel quality than straight-chain alkanes.

## 6.8 Aromatisation

Alkanes with six or more carbon atoms can form benzene and its homologues on heating under suitable conditions.

Conditions:

- about `773 K`
- pressure about `10-20 atm`
- oxides of vanadium, molybdenum, or chromium supported on alumina

Example:

```text
n-Hexane --heat/catalyst--> Benzene + 4H2
```

This is also called **reforming**.

## 6.9 Reaction with Steam

Methane reacts with steam at high temperature in presence of nickel catalyst to form carbon monoxide and hydrogen.

```text
CH4 + H2O --Ni, heat--> CO + 3H2
```

This is industrially important for preparing hydrogen gas.

## 6.10 Pyrolysis or Cracking

Higher alkanes decompose into smaller alkanes and alkenes at high temperature.

This is called:

- pyrolysis
- cracking

Example:

```text
C12H26 --973 K, Pt/Pd/Ni--> C7H16 + C5H10 + other products
Dodecane                    Heptane  Pentene
```

Importance:

Cracking converts heavier petroleum fractions into more useful lighter fuels and alkenes.

# 7. Conformations of Alkanes

## 7.1 Meaning of Conformation

Conformations are different spatial arrangements produced by rotation around a carbon-carbon single bond.

They are also called:

- conformers
- conformational isomers
- rotamers

Important:

Conformations interconvert without breaking bonds.

## 7.2 Why Rotation Around C-C Bond Is Possible

A C-C single bond is a sigma bond.

The electron cloud of a sigma bond is symmetrical around the internuclear axis.

So rotation around the bond does not destroy overlap.

Therefore, rotation is possible.

However, rotation is not completely free because repulsion between bond electron clouds creates a small energy barrier.

This repulsion is called **torsional strain**.

## 7.3 Ethane as Example

Ethane:

`CH3-CH3`

Each carbon has three hydrogen atoms.

If one carbon is kept fixed and the other is rotated around the C-C bond, many arrangements are possible.

Two extreme conformations are important:

| Conformation | Arrangement | Stability |
| --- | --- | --- |
| Eclipsed | Hydrogens on front and rear carbon are closest | Less stable |
| Staggered | Hydrogens are farthest apart | More stable |

## 7.4 Eclipsed Conformation

In eclipsed conformation:

- C-H bonds on front carbon line up with C-H bonds on rear carbon
- electron clouds are close
- repulsion is high
- torsional strain is maximum
- energy is high
- stability is low

## 7.5 Staggered Conformation

In staggered conformation:

- C-H bonds are far apart
- electron cloud repulsion is minimum
- torsional strain is minimum
- energy is low
- stability is maximum

## 7.6 Skew Conformations

Any intermediate conformation between eclipsed and staggered forms is called a skew conformation.

## 7.7 Important Numerical Point

The energy difference between eclipsed and staggered ethane is about:

`12.5 kJ mol-1`

This is small.

Therefore, at ordinary temperature, ethane molecules can rotate around the C-C bond almost freely for practical purposes.

## 7.8 Sawhorse and Newman Projections

Ethane conformations are commonly represented by:

| Projection | How It Views the Molecule |
| --- | --- |
| Sawhorse projection | Viewed along the molecular axis but drawn with an inclined C-C bond. |
| Newman projection | Viewed directly along the C-C bond. |

In Newman projection:

- front carbon is shown as a point
- rear carbon is shown as a circle
- bonds from front carbon start from the centre
- bonds from rear carbon start from the circle

## 7.9 Exam Answer Frame: Why Staggered Ethane Is More Stable

Write:

Staggered ethane is more stable than eclipsed ethane because the C-H bond electron clouds on adjacent carbon atoms are farthest apart in the staggered form. This minimises repulsive interactions and torsional strain. In eclipsed ethane, bond electron clouds are closest, so repulsion and energy are maximum.

# 8. Alkenes

## 8.1 Meaning

Alkenes are unsaturated hydrocarbons containing at least one carbon-carbon double bond.

General formula for open-chain monoalkenes:

`CnH2n`

Examples:

| Formula | Name |
| --- | --- |
| `C2H4` | Ethene |
| `C3H6` | Propene |
| `C4H8` | Butene |

Alkenes are also called **olefins** because ethene was known to form an oily liquid with chlorine.

## 8.2 Structure of the Double Bond

A carbon-carbon double bond contains:

- one sigma bond
- one pi bond

In ethene:

- each carbon is `sp2` hybridised
- molecule is planar
- bond angle is about `120°`
- C=C bond length is about `134 pm`

The sigma bond forms by head-on overlap of `sp2` orbitals.

The pi bond forms by sideways overlap of unhybridised p orbitals.

## 8.3 Why Alkenes Are Reactive

The pi bond is weaker than the sigma bond.

It contains loosely held mobile electrons above and below the plane of the molecule.

Therefore, alkenes are attacked by electrophiles.

Main reaction type:

**electrophilic addition**

## 8.4 Naming Alkenes

Rules:

1. Choose the longest chain containing the double bond.
2. Number from the end nearer to the double bond.
3. Replace `ane` of alkane with `ene`.
4. Mention the position of double bond.
5. Name substituents with locants.

Examples:

| Structure | IUPAC Name |
| --- | --- |
| `CH2=CH2` | Ethene |
| `CH3-CH=CH2` | Propene |
| `CH3-CH2-CH=CH2` | But-1-ene |
| `CH3-CH=CH-CH3` | But-2-ene |
| `CH2=CH-CH=CH2` | Buta-1,3-diene |
| `CH2=C(CH3)-CH3` | 2-Methylprop-1-ene |

## 8.5 Counting Sigma and Pi Bonds

Useful rules:

| Bond | Sigma Bonds | Pi Bonds |
| --- | --- | --- |
| Single bond | 1 | 0 |
| Double bond | 1 | 1 |
| Triple bond | 1 | 2 |

Example:

Ethene, `CH2=CH2`:

- C-H bonds = 4 sigma
- C=C = 1 sigma + 1 pi
- total sigma = 5
- total pi = 1

## 8.6 Structural Isomerism in Alkenes

Alkenes show:

- chain isomerism
- position isomerism

Example: `C4H8`

| Structure | Name | Type Relation |
| --- | --- | --- |
| `CH2=CH-CH2-CH3` | But-1-ene | position isomer of but-2-ene |
| `CH3-CH=CH-CH3` | But-2-ene | position isomer of but-1-ene |
| `CH2=C(CH3)-CH3` | 2-Methylprop-1-ene | chain isomer |

## 8.7 Geometrical Isomerism

Geometrical isomerism occurs because rotation around C=C is restricted.

For cis-trans isomerism, each double-bond carbon must have two different groups attached.

### Cis Isomer

Identical or similar groups are on the same side of the double bond.

### Trans Isomer

Identical or similar groups are on opposite sides of the double bond.

Example:

But-2-ene has cis and trans forms.

| Isomer | Arrangement | Dipole |
| --- | --- | --- |
| cis-But-2-ene | `CH3` groups on same side | Polar, dipole does not cancel fully |
| trans-But-2-ene | `CH3` groups on opposite sides | Nearly non-polar, dipoles cancel |

## 8.8 Conditions for Cis-Trans Isomerism

A compound shows geometrical isomerism if:

1. It contains a C=C bond.
2. Rotation around C=C is restricted.
3. Each carbon of C=C is attached to two different groups.

Examples:

| Compound | Shows Cis-Trans? | Reason |
| --- | --- | --- |
| `CHCl=CHCl` | Yes | each C has H and Cl |
| `CH2=CBr2` | No | one carbon has two H atoms and other has two Br atoms |
| `C6H5CH=CHCH3` | Yes | each C has two different groups |
| `CH3CH=CClCH3` | Yes | each C has two different groups |

## 8.9 Why Cis and Trans Isomers Differ in Properties

They have:

- same molecular formula
- same structural connectivity
- different spatial arrangement

Because of this, they may differ in:

- boiling point
- melting point
- dipole moment
- solubility
- stability

For solids, trans isomers often have higher melting points because they are more symmetrical and pack better in crystals.

# 9. Preparation of Alkenes

## 9.1 From Alkynes: Partial Reduction

Alkynes can be partially reduced to alkenes.

### With Lindlar's Catalyst

Lindlar's catalyst is partially deactivated palladised charcoal.

It gives cis-alkenes.

Example:

```text
HC#CH + H2 --Lindlar catalyst--> CH2=CH2
```

For suitable substituted alkynes, the product has cis geometry.

### With Sodium in Liquid Ammonia

Alkynes on reduction with sodium in liquid ammonia give trans-alkenes.

Comparison:

| Reagent | Product Geometry |
| --- | --- |
| Lindlar catalyst | cis-alkene |
| Na/liquid NH3 | trans-alkene |

## 9.2 From Alkyl Halides: Dehydrohalogenation

Alkyl halides form alkenes on heating with alcoholic KOH.

This removes one molecule of hydrogen halide.

General idea:

```text
R-CH2-CH2-X --alc. KOH, heat--> R-CH=CH2 + HX
```

This is a beta-elimination reaction.

Meaning:

- halogen leaves from alpha carbon
- hydrogen leaves from beta carbon
- double bond forms between alpha and beta carbons

Rate order for halogen:

`I > Br > Cl`

Rate order for alkyl halide:

`tertiary > secondary > primary`

## 9.3 From Vicinal Dihalides: Dehalogenation

Vicinal dihalides have two halogen atoms on adjacent carbon atoms.

They react with zinc to form alkenes.

Example:

```text
CH2Br-CH2Br + Zn -> CH2=CH2 + ZnBr2
```

This is dehalogenation because halogen atoms are removed.

## 9.4 From Alcohols: Acidic Dehydration

Alcohols form alkenes on heating with concentrated sulphuric acid.

General idea:

```text
R-CH2-CH2OH --conc. H2SO4, heat--> R-CH=CH2 + H2O
```

This is called acidic dehydration because water is removed.

It is also a beta-elimination reaction.

# 11. Chemical Properties of Alkenes

Alkenes mainly show addition reactions because of the pi bond.

Main reactions:

1. Addition of hydrogen.
2. Addition of halogens.
3. Addition of hydrogen halides.
4. Addition of sulphuric acid.
5. Addition of water.
6. Oxidation.
7. Ozonolysis.
8. Polymerisation.

---

## 11.1 Addition of Hydrogen

Alkenes add hydrogen in the presence of nickel, palladium, or platinum to form alkanes.

Example:

```text
CH2=CH2 + H2 --Ni/Pd/Pt--> CH3-CH3
```

This is hydrogenation.

## 11.2 Addition of Halogens

Alkenes add chlorine or bromine to form vicinal dihalides.

Example:

```text
CH2=CH2 + Br2 -> BrCH2-CH2Br
Ethene             1,2-dibromoethane
```

For propene:

```text
CH3-CH=CH2 + Cl2 -> CH3-CHCl-CH2Cl
Propene              1,2-dichloropropane
```

### Test for Unsaturation

Bromine in carbon tetrachloride is reddish orange.

Alkenes decolourise bromine solution because bromine adds across the double bond.

This is used as a test for unsaturation.

## 11.3 Addition of Hydrogen Halides

Hydrogen halides add to alkenes to form alkyl halides.

Reactivity order:

`HI > HBr > HCl`

Example with ethene:

```text
CH2=CH2 + HBr -> CH3-CH2Br
```

Because ethene is symmetrical, only one product is possible.

## 11.4 Markovnikov Rule

For unsymmetrical alkenes, addition of HX follows Markovnikov rule.

Rule:

**The negative part of the addendum attaches to the double-bond carbon that already has fewer hydrogen atoms.**

Simpler learner version:

**H goes to the carbon already having more H; X goes to the carbon having fewer H.**

Example:

```text
CH3-CH=CH2 + HBr -> CH3-CHBr-CH3
Propene              2-bromopropane
```

## 11.5 Why Markovnikov Rule Works

Addition happens through carbocation formation.

In propene:

If H+ adds to terminal carbon, the carbocation forms on middle carbon:

```text
CH3-CH+-CH3
```

This is a secondary carbocation.

If H+ adds to middle carbon, the carbocation forms on terminal carbon:

```text
CH3-CH2-CH2+
```

This is a primary carbocation.

Secondary carbocation is more stable than primary carbocation.

Therefore, the reaction follows the path forming the more stable carbocation.

Then `Br-` attacks the carbocation.

Product:

`2-bromopropane`

## 11.6 Peroxide Effect or Kharash Effect

In the presence of peroxide, HBr adds to unsymmetrical alkenes opposite to Markovnikov rule.

This is called:

- peroxide effect
- Kharash effect
- anti-Markovnikov addition

Example:

```text
CH3-CH=CH2 + HBr --peroxide--> CH3-CH2-CH2Br
Propene                         1-bromopropane
```

Important:

This effect is observed with HBr only, not with HCl or HI.

## 11.7 Why Peroxide Effect Gives Anti-Markovnikov Product

In peroxide effect, the mechanism is free radical, not carbocation.

The bromine radical adds to the double bond in such a way that the more stable carbon radical forms.

For propene, formation of a secondary free radical is favoured.

Then that radical abstracts hydrogen from HBr.

This produces `1-bromopropane`.

## 11.8 Why Peroxide Effect Is Not Seen with HCl and HI

It depends on bond energies and radical chain feasibility.

| Hydrogen Halide | Reason |
| --- | --- |
| HCl | H-Cl bond is too strong; radical chain step is not favourable. |
| HBr | Bond energy and radical steps are suitable. |
| HI | H-I bond is weak, but iodine radicals tend to combine and chain does not proceed effectively. |

## 11.9 Addition of Sulphuric Acid

Cold concentrated sulphuric acid adds to alkenes according to Markovnikov rule to form alkyl hydrogen sulphates.

Example idea:

```text
CH2=CH2 + H2SO4 -> CH3-CH2-OSO3H
Ethene             ethyl hydrogen sulphate
```

## 11.10 Addition of Water: Hydration

Alkenes add water in the presence of a few drops of concentrated sulphuric acid to form alcohols.

The addition follows Markovnikov rule.

Example:

```text
CH3-CH=CH2 + H2O --H2SO4--> CH3-CH(OH)-CH3
Propene                         Propan-2-ol
```

## 11.11 Oxidation with Cold Alkaline KMnO4

Cold, dilute, aqueous potassium permanganate oxidises alkenes to vicinal glycols.

This reagent is called Baeyer's reagent.

Example:

```text
CH2=CH2 --cold alkaline KMnO4--> HO-CH2-CH2-OH
Ethene                              Ethane-1,2-diol
```

The purple colour of KMnO4 is discharged.

This is another test for unsaturation.

## 11.12 Strong Oxidation

Acidic KMnO4 or acidic K2Cr2O7 can cleave alkenes to acids and/or ketones depending on the structure.

Example:

```text
CH3-CH=CH-CH3 --KMnO4/H+--> 2 CH3COOH
But-2-ene                      Ethanoic acid
```

## 11.13 Ozonolysis

Ozonolysis is a reaction of alkenes with ozone followed by cleavage of the ozonide.

Typical workup:

`Zn/H2O`

Use:

**Ozonolysis helps locate the position of the double bond.**

Example:

```text
CH3-CH=CH-CH3 --O3/Zn-H2O--> 2 CH3CHO
But-2-ene                      Ethanal
```

How to think:

1. Cut the double bond.
2. Put oxygen on each double-bond carbon.
3. Identify aldehydes or ketones formed.

## 11.14 Polymerisation

Many alkene molecules join to form a large molecule called a polymer.

The small molecule is called a monomer.

Ethene gives polythene:

```text
n CH2=CH2 --heat/pressure/catalyst--> -(CH2-CH2)n-
```

Propene gives polypropene:

```text
n CH3-CH=CH2 --heat/pressure/catalyst--> -(CH(CH3)-CH2)n-
```

Uses:

| Polymer | Uses |
| --- | --- |
| Polythene | bags, sheets, squeeze bottles, containers |
| Polypropene | buckets, crates, moulded articles |

Environmental note:

Excessive use of polythene and polypropylene creates waste-management problems.

# 12. Alkynes

## 12.1 Meaning

Alkynes are unsaturated hydrocarbons containing at least one carbon-carbon triple bond.

General formula for open-chain monoalkynes:

`CnH2n-2`

First stable member:

`HC#CH`

Common name:

Acetylene.

IUPAC name:

Ethyne.

## 12.2 Importance of Ethyne

Ethyne is used in oxyacetylene flame for welding.

Reason:

It burns with oxygen to produce a very hot flame.

Alkynes are also useful starting materials for many organic compounds.

## 12.3 Naming Alkynes

Rules:

1. Choose the longest chain containing the triple bond.
2. Number from the end nearer to the triple bond.
3. Replace `ane` with `yne`.
4. Mention position of triple bond.

Examples:

| Structure | IUPAC Name |
| --- | --- |
| `HC#CH` | Ethyne |
| `CH3-C#CH` | Propyne |
| `HC#C-CH2-CH3` | But-1-yne |
| `CH3-C#C-CH3` | But-2-yne |
| `HC#C-CH2-CH2-CH3` | Pent-1-yne |
| `CH3-C#C-CH2-CH3` | Pent-2-yne |

## 12.4 Isomerism in Alkynes

Ethyne and propyne have only one structure.

From butyne onward, position isomerism is possible.

Example:

`C4H6`

| Structure | Name |
| --- | --- |
| `HC#C-CH2-CH3` | But-1-yne |
| `CH3-C#C-CH3` | But-2-yne |

For `C5H8`, both position and chain isomerism are possible.

Examples:

| Structure | Name |
| --- | --- |
| `HC#C-CH2-CH2-CH3` | Pent-1-yne |
| `CH3-C#C-CH2-CH3` | Pent-2-yne |
| `CH3-CH(CH3)-C#CH` | 3-Methylbut-1-yne |

## 12.5 Structure of Triple Bond

In ethyne:

- each carbon is `sp` hybridised
- molecule is linear
- H-C-C bond angle is `180°`
- C#C bond contains one sigma bond and two pi bonds

Bond comparison:

| Bond | Approximate Bond Length | Approximate Bond Enthalpy |
| --- | --- | --- |
| C-C | `154 pm` | `348 kJ mol-1` |
| C=C | `133-134 pm` | `681 kJ mol-1` |
| C#C | `120 pm` | `823 kJ mol-1` |

Important:

The triple bond is shorter and stronger than double and single bonds, but it still has pi electrons, so alkynes undergo addition reactions.

## 12.6 Sigma and Pi Bonds in Ethyne

Ethyne structure:

`H-C#C-H`

Bonds:

- two C-H sigma bonds
- one C-C sigma bond
- two C-C pi bonds

Total:

- sigma bonds = 3
- pi bonds = 2

# 15. Chemical Properties of Alkynes

Alkynes show:

1. Acidic character.
2. Addition reactions.
3. Polymerisation.

---

## 15.1 Acidic Character of Terminal Alkynes

Terminal alkynes contain hydrogen attached directly to a triply bonded carbon.

Example:

`HC#CH`

This hydrogen is acidic compared with hydrogens in alkenes and alkanes.

Reason:

- carbon in terminal alkyne is `sp` hybridised
- `sp` carbon has 50% s-character
- higher s-character means higher electronegativity
- carbon attracts C-H bond electrons more strongly
- H can be released more easily as `H+`

Acidity order:

`ethyne > ethene > ethane`

or in general:

`sp C-H > sp2 C-H > sp3 C-H`

## 15.2 Reaction with Sodium

Ethyne reacts with sodium to form sodium acetylide and hydrogen.

```text
HC#CH + Na -> HC#C-Na + 1/2 H2
```

Further reaction:

```text
HC#C-Na + Na -> Na-C#C-Na + 1/2 H2
```

## 15.3 Reaction with Sodamide

Terminal alkynes react with sodamide to form sodium alkynides.

Example:

```text
CH3-C#CH + NaNH2 -> CH3-C#C-Na + NH3
```

This reaction is used to distinguish terminal alkynes from alkenes and alkanes.

Important:

Internal alkynes do not have acidic terminal hydrogen, so they do not show this reaction in the same way.

## 15.4 Addition of Hydrogen

Alkynes add hydrogen to form alkenes and then alkanes.

Full hydrogenation:

```text
HC#CH + 2H2 --Ni/Pd/Pt--> CH3-CH3
```

Partial hydrogenation:

```text
alkyne + H2 --Lindlar catalyst--> cis-alkene
```

## 15.5 Addition of Halogens

Alkynes add bromine or chlorine.

Example:

```text
HC#CH + Br2 -> BrCH=CHBr
```

Further addition:

```text
BrCH=CHBr + Br2 -> CHBr2-CHBr2
```

Bromine solution is decolourised, so alkynes also test positive for unsaturation.

## 15.6 Addition of Hydrogen Halides

Hydrogen halides add to alkynes.

Two molecules of HX may add to form geminal dihalides.

Geminal dihalide:

Two halogen atoms are attached to the same carbon atom.

Example:

```text
HC#CH + HBr -> CH2=CHBr
CH2=CHBr + HBr -> CH3-CHBr2
```

Product:

`1,1-dibromoethane`

For unsymmetrical alkynes, addition follows Markovnikov rule.

## 15.7 Addition of Water

Alkynes do not react with water under ordinary conditions.

They add water when warmed with:

- mercuric sulphate, `HgSO4`
- dilute sulphuric acid, `H2SO4`
- about `333 K`

Ethyne gives ethanal:

```text
HC#CH + H2O --HgSO4/H2SO4--> CH3CHO
```

Propyne gives propanone:

```text
CH3-C#CH + H2O --HgSO4/H2SO4--> CH3COCH3
```

Reason:

The first product is an enol, which rearranges to a more stable carbonyl compound.

## 15.8 Linear Polymerisation of Ethyne

Ethyne can polymerise linearly to give polyacetylene.

Representation:

```text
n HC#CH -> -(CH=CH)n-
```

Polyacetylene can conduct electricity under suitable conditions.

Thin films of polyacetylene can be used as electrodes in batteries.

## 15.9 Cyclic Polymerisation of Ethyne

When ethyne is passed through a red-hot iron tube at about `873 K`, three molecules combine to form benzene.

```text
3 HC#CH --red hot Fe tube, 873 K--> C6H6
```

This reaction connects aliphatic compounds to aromatic compounds.

# 17. Structure of Benzene

## 17.1 Formula and Problem

Benzene has molecular formula:

`C6H6`

This formula suggests a high degree of unsaturation.

Yet benzene does not behave like ordinary alkenes.

It does not easily undergo addition reactions under normal conditions.

This unusual stability required special explanation.

## 17.2 Kekule Structure

Kekule proposed a cyclic structure with:

- six carbon atoms in a ring
- alternating single and double bonds
- one hydrogen attached to each carbon

But this structure had problems.

If single and double bonds were fixed, benzene should give two different ortho disubstituted products.

Experimentally, benzene gives only one ortho disubstituted product.

## 17.3 Resonance Explanation

Benzene is not one fixed structure with three localised double bonds.

It is a resonance hybrid of equivalent contributing structures.

The six pi electrons are delocalised over the whole ring.

This delocalisation gives benzene unusual stability.

## 17.4 Orbital Picture of Benzene

In benzene:

- all six carbon atoms are `sp2` hybridised
- the ring is planar
- each carbon forms two C-C sigma bonds
- each carbon forms one C-H sigma bond
- each carbon has one unhybridised p orbital

The p orbitals overlap sideways around the ring.

This forms a continuous delocalised pi electron cloud above and below the plane of the ring.

## 17.5 Bond Length Evidence

If benzene had fixed single and double bonds:

- C-C single bonds should be about `154 pm`
- C=C double bonds should be about `133 pm`

Actual benzene:

- all C-C bond lengths are equal
- each is about `139 pm`

This is intermediate between single and double bond lengths.

Conclusion:

Benzene has delocalised bonds, not fixed alternating single and double bonds.

## 17.6 Why Benzene Prefers Substitution

Addition reaction would destroy aromatic delocalisation.

Substitution reaction replaces a hydrogen but keeps the aromatic ring intact.

Therefore:

**Benzene undergoes electrophilic substitution rather than ordinary addition under normal conditions.**

# 18. Aromaticity

## 18.1 Conditions for Aromaticity

A system is aromatic if it satisfies these conditions:

1. It is cyclic.
2. It is planar.
3. It has continuous delocalisation of pi electrons.
4. It follows Huckel rule: `(4n + 2)` pi electrons, where `n = 0, 1, 2, ...`.

## 18.2 Huckel Rule

Number of pi electrons must be:

`4n + 2`

Allowed values:

| n | Pi Electrons |
| --- | --- |
| 0 | 2 |
| 1 | 6 |
| 2 | 10 |
| 3 | 14 |

Benzene has 6 pi electrons.

For benzene:

`4n + 2 = 6`

`n = 1`

So benzene satisfies Huckel rule.

## 18.3 Why Some Cyclic Compounds Are Not Aromatic

A compound is not aromatic if any condition fails.

Common failures:

| Failure | Result |
| --- | --- |
| Not cyclic | Not aromatic |
| Not planar | Poor delocalisation |
| No continuous p orbital overlap | Not aromatic |
| Has `4n` pi electrons instead of `4n + 2` | Anti-aromatic or non-aromatic depending on case |

# 21. Chemical Properties of Aromatic Hydrocarbons

Arenes mainly undergo electrophilic substitution reactions.

Common reactions:

1. Nitration.
2. Halogenation.
3. Sulphonation.
4. Friedel-Crafts alkylation.
5. Friedel-Crafts acylation.

They can also undergo addition under vigorous conditions.

---

## 21.1 Nitration of Benzene

Benzene reacts with a mixture of concentrated nitric acid and concentrated sulphuric acid to form nitrobenzene.

```text
C6H6 + HNO3 --conc. H2SO4--> C6H5NO2 + H2O
```

Electrophile:

`NO2+`, nitronium ion.

Product:

Nitrobenzene.

## 21.2 Halogenation of Benzene

Benzene reacts with chlorine or bromine in presence of Lewis acid catalyst.

Catalysts:

- anhydrous `FeCl3`
- `FeBr3`
- `AlCl3`

Example:

```text
C6H6 + Cl2 --FeCl3--> C6H5Cl + HCl
```

Product:

Chlorobenzene.

## 21.3 Sulphonation of Benzene

Benzene reacts with fuming sulphuric acid or oleum to form benzene sulphonic acid.

```text
C6H6 + SO3 --H2SO4--> C6H5SO3H
```

This reaction introduces the `-SO3H` group.

## 21.4 Friedel-Crafts Alkylation

Benzene reacts with alkyl halide in presence of anhydrous aluminium chloride to form alkylbenzene.

Example:

```text
C6H6 + CH3Cl --anhyd. AlCl3--> C6H5CH3 + HCl
```

Product:

Toluene.

Important:

Alkyl carbocations may rearrange. Therefore, treating benzene with 1-chloropropane may give isopropylbenzene rather than n-propylbenzene.

## 21.5 Friedel-Crafts Acylation

Benzene reacts with acyl halide or acid anhydride in presence of Lewis acid to form acylbenzene.

Example:

```text
C6H6 + CH3COCl --anhyd. AlCl3--> C6H5COCH3 + HCl
```

Product:

Acetophenone.

Friedel-Crafts acylation is useful because it introduces an acyl group without carbocation rearrangement problems typical of alkylation.

## 21.6 Excess Substitution

If excess electrophile is used, further substitution can occur.

Example:

Benzene with excess chlorine in presence of anhydrous aluminium chloride can produce highly chlorinated benzene derivatives such as hexachlorobenzene.

# 22. Mechanism of Electrophilic Aromatic Substitution

Electrophilic aromatic substitution is often written as `SE` reaction:

- `S` = substitution
- `E` = electrophilic

There are three major stages:

1. Generation of electrophile.
2. Formation of sigma complex/arenium ion.
3. Removal of proton and restoration of aromaticity.

## 22.1 Stage 1: Generation of Electrophile

Examples:

| Reaction | Electrophile |
| --- | --- |
| Nitration | `NO2+` |
| Chlorination | `Cl+` equivalent |
| Friedel-Crafts alkylation | `R+` |
| Friedel-Crafts acylation | `RCO+` acylium ion |

In nitration:

Sulphuric acid protonates nitric acid, which then loses water to form nitronium ion.

```text
HNO3 + H2SO4 -> NO2+ + HSO4- + H2O
```

## 22.2 Stage 2: Formation of Sigma Complex

The electrophile attacks the benzene ring.

One pi bond helps form a new sigma bond with electrophile.

This gives a carbocation intermediate called:

- sigma complex
- arenium ion

Important:

The sigma complex is resonance-stabilised, but it is not aromatic because one carbon becomes `sp3` hybridised and continuous delocalisation is interrupted.

## 22.3 Stage 3: Loss of Proton

A base removes the proton from the carbon bearing the electrophile.

Aromaticity is restored.

This step gives the substituted benzene product.

## 22.4 Why Restoration of Aromaticity Matters

Aromaticity gives strong stability.

The reaction path is favourable because final product regains aromatic character.

This is why benzene substitutes rather than adds.

# 24. Directive Influence in Monosubstituted Benzene

## 24.1 Meaning

When a monosubstituted benzene undergoes further substitution, the incoming group may enter:

- ortho position
- meta position
- para position

The existing substituent controls which position is favoured.

This is called **directive influence**.

## 24.2 Two Questions to Ask

For any substituent already on benzene, ask:

1. Does it activate or deactivate the ring?
2. Does it direct the incoming group to ortho/para or meta positions?

## 24.3 Ortho-Para Directing Groups

Ortho-para directing groups direct incoming electrophile to ortho and para positions.

Common examples:

| Group | Effect |
| --- | --- |
| `-OH` | strongly activating, ortho/para directing |
| `-NH2` | strongly activating, ortho/para directing |
| `-NHR` | activating, ortho/para directing |
| `-NHCOCH3` | activating, ortho/para directing |
| `-OCH3` | activating, ortho/para directing |
| `-CH3` | activating, ortho/para directing |
| `-C2H5` | activating, ortho/para directing |

## 24.4 Why `-OH` Directs Ortho/Para

The `-OH` group donates electron density to the benzene ring through resonance.

This increases electron density mainly at ortho and para positions.

Electrophiles attack electron-rich positions.

Therefore, phenol gives mainly ortho and para substituted products.

## 24.5 Halogens: Special Case

Halogens are:

- deactivating
- ortho/para directing

Why deactivating?

Halogens withdraw electrons by strong `-I` effect.

Why ortho/para directing?

They donate electron density by resonance to ortho and para positions.

So halogens make the ring less reactive overall, but still direct incoming electrophiles to ortho and para positions.

## 24.6 Meta Directing Groups

Meta directing groups direct incoming electrophile to meta position.

Common examples:

| Group | Nature |
| --- | --- |
| `-NO2` | strongly deactivating |
| `-CN` | deactivating |
| `-CHO` | deactivating |
| `-COR` | deactivating |
| `-COOH` | deactivating |
| `-COOR` | deactivating |
| `-SO3H` | deactivating |

## 24.7 Why Nitro Group Directs Meta

The `-NO2` group strongly withdraws electron density by `-I` and resonance effects.

This reduces electron density especially at ortho and para positions.

The meta position becomes relatively more favourable for electrophilic attack.

Therefore, nitrobenzene gives mainly meta-substituted product.

## 24.8 Reactivity Order Example

For nitration:

Toluene reacts faster than benzene because `-CH3` activates the ring.

Nitrobenzene reacts slower than benzene because `-NO2` deactivates the ring.

So:

`toluene > benzene > nitrobenzene`

# 25. Carcinogenicity and Toxicity

Benzene and many polynuclear aromatic hydrocarbons are toxic.

Polynuclear aromatic hydrocarbons contain more than two fused benzene rings.

They can be formed during incomplete combustion of organic materials such as:

- tobacco
- coal
- petroleum
- other carbon-rich fuels

These compounds can enter the body and undergo biochemical changes.

Some may damage DNA and cause cancer.

Important exam sentence:

**Benzene and fused polynuclear aromatic hydrocarbons can be toxic and carcinogenic; incomplete combustion of organic matter is one source of such compounds.**

# 26. Reaction Logic: How to Predict Products

## 26.1 If the Compound Is an Alkane

Ask:

1. Is there light/heat and halogen?
   - Expect free radical substitution.
2. Is there oxygen?
   - Expect combustion.
3. Is there controlled oxidation?
   - Expect alcohol/aldehyde/acid depending conditions.
4. Is there high heat?
   - Expect cracking/pyrolysis.
5. Is there reforming catalyst?
   - Expect aromatisation for suitable long-chain alkanes.

## 26.2 If the Compound Is an Alkene

Ask:

1. Is reagent `H2/Ni`, `H2/Pd`, or `H2/Pt`?
   - Alkane.
2. Is reagent `Br2/CCl4` or `Cl2`?
   - Vicinal dihalide.
3. Is reagent `HX`?
   - Alkyl halide, Markovnikov product unless peroxide with HBr.
4. Is reagent `H2O/H+`?
   - Alcohol, Markovnikov addition.
5. Is reagent cold alkaline `KMnO4`?
   - Vicinal glycol.
6. Is reagent `O3/Zn-H2O`?
   - Carbonyl fragments, locate double bond.
7. Are heat, pressure, catalyst used with many monomers?
   - Polymer.

## 26.3 If the Compound Is an Alkyne

Ask:

1. Is it terminal?
   - It can show acidic reactions with sodium or sodamide.
2. Is hydrogenation complete?
   - Alkane.
3. Is hydrogenation partial?
   - Alkene.
4. Is `HX` added?
   - Geminal dihalide after two equivalents.
5. Is water added with `HgSO4/H2SO4`?
   - Carbonyl compound.
6. Is ethyne passed through red-hot iron tube?
   - Benzene.

## 26.4 If the Compound Is Benzene or Arene

Ask:

1. Is there `HNO3/H2SO4`?
   - Nitration.
2. Is there `Cl2/FeCl3` or `Br2/FeBr3`?
   - Halogenation.
3. Is there fuming `H2SO4`?
   - Sulphonation.
4. Is there alkyl halide/AlCl3?
   - Friedel-Crafts alkylation.
5. Is there acyl chloride/AlCl3?
   - Friedel-Crafts acylation.
6. Is there an existing substituent?
   - Apply directive influence.

# 27. Markovnikov and Peroxide Effect: Detailed Comparison

| Feature | Markovnikov Addition | Peroxide Effect |
| --- | --- | --- |
| Reagent | HX | HBr with peroxide |
| Mechanism | Carbocation/electrophilic addition | Free radical chain |
| Major intermediate | More stable carbocation | More stable free radical |
| Product orientation | X to carbon with fewer H | Br to carbon with more H in many terminal alkenes |
| Example with propene | 2-bromopropane | 1-bromopropane |
| Applies to | HCl, HBr, HI generally | HBr only |

## 27.1 How to Answer a Mechanism Question

For normal HBr addition to propene:

1. State that HBr gives `H+` and `Br-`.
2. The pi bond attacks `H+`.
3. More stable secondary carbocation forms.
4. `Br-` attacks carbocation.
5. Major product is 2-bromopropane.

For peroxide effect:

1. Peroxide generates free radicals.
2. Bromine radical forms.
3. Bromine radical adds to propene to form more stable secondary radical.
4. The radical abstracts hydrogen from HBr.
5. Major product is 1-bromopropane.

# 28. Ozonolysis: How to Work Backward

Ozonolysis is often used in structure determination.

## 28.1 Forward Method

Given alkene, find products:

1. Identify C=C.
2. Cut the double bond.
3. Put `=O` on each double-bond carbon.
4. Name the carbonyl products.

Example:

```text
CH3-CH=CH-CH3 -> CH3CHO + CH3CHO
```

But-2-ene gives two ethanal molecules.

## 28.2 Backward Method

Given ozonolysis products, find alkene:

1. Remove the oxygen from each carbonyl carbon.
2. Join those two carbons by a double bond.
3. Keep the remaining groups attached.

Example:

Products:

- ethanal, `CH3CHO`
- pentan-3-one, `CH3CH2COCH2CH3`

The original alkene must have had these two carbonyl carbons joined by C=C.

This method is useful for exercise problems.

# 29. Acidic Behaviour: Ethyne, Benzene, n-Hexane

A common comparison question asks:

Arrange benzene, n-hexane, and ethyne in decreasing acidic behaviour.

Answer:

`ethyne > benzene > n-hexane`

Reason:

| Compound | Relevant Carbon | Acidic Character |
| --- | --- | --- |
| Ethyne | `sp` carbon | Most acidic due to 50% s-character |
| Benzene | `sp2` carbon | Less acidic than ethyne |
| n-Hexane | `sp3` carbon | Least acidic |

Higher s-character means carbon holds electrons more strongly and stabilises the conjugate base better.

# 30. Benzene Stability: Answer Frame

If asked, "Why is benzene extraordinarily stable though it contains three double bonds?", write:

Benzene is not a molecule with three isolated double bonds. All six carbon atoms are `sp2` hybridised and the unhybridised p orbitals overlap continuously around the ring. The six pi electrons are delocalised above and below the plane of the ring. This delocalisation gives resonance stabilisation. X-ray data shows that all C-C bond lengths in benzene are equal, about `139 pm`, intermediate between single and double bonds. Because of this aromatic stabilisation, benzene is unusually stable and prefers substitution reactions that retain aromaticity.

# 32. Common Mistakes and Corrections

| Mistake | Correct Understanding |
| --- | --- |
| Thinking all unsaturated compounds behave alike | Benzene is unsaturated by formula but aromatic and unusually stable. |
| Choosing the longest visible line as parent chain | Choose the longest continuous chain containing the multiple bond if present. |
| Forgetting double bond priority in numbering alkenes | Number from the end closer to C=C. |
| Applying peroxide effect to HCl or HI | Peroxide effect is characteristic of HBr only. |
| Saying alkanes never react | They react under suitable conditions such as UV halogenation or combustion. |
| Treating all alkyne hydrogens as acidic | Only hydrogens attached to triply bonded carbon are acidic. |
| Saying halogens are activating in benzene | Halogens are deactivating but ortho/para directing. |
| Confusing vicinal and geminal dihalides | Vicinal = halogens on adjacent carbons; geminal = halogens on same carbon. |
| Forgetting aromaticity conditions | Planar, cyclic, continuous delocalisation, `(4n + 2)` pi electrons. |

# 33. NCERT Exercise Guidance

This section does not replace solving the exercises. It tells you how to approach each type.

## 33.1 Formation of Ethane During Chlorination of Methane

Key idea:

Ethane forms when two methyl radicals combine during termination.

Equation:

```text
CH3. + CH3. -> CH3-CH3
```

Use this to answer questions about side products in radical halogenation.

## 33.2 IUPAC Naming Questions

Checklist:

1. Identify parent chain.
2. Include double/triple bond in parent chain if present.
3. Number from the correct end.
4. Name substituents alphabetically.
5. Use `ene` or `yne` with locant.
6. Check whether benzene derivative needs ortho/meta/para or numerical locants.

## 33.3 Isomer Questions

For alkene isomers:

1. Draw straight-chain forms.
2. Move the double bond.
3. Create branched forms.
4. Check for duplicate structures.
5. Add cis/trans only if asked and if conditions are satisfied.

For alkyne isomers:

1. Draw straight-chain forms.
2. Move triple bond.
3. Add branched skeletons.
4. Remember terminal and internal alkynes.

## 33.4 Combustion Questions

Use:

```text
CxHy + (x + y/4)O2 -> xCO2 + (y/2)H2O
```

If fractions appear, multiply whole equation by 2.

Example:

For pentene, `C5H10`:

```text
C5H10 + 15/2 O2 -> 5CO2 + 5H2O
```

Multiply by 2:

```text
2C5H10 + 15O2 -> 10CO2 + 10H2O
```

## 33.5 Cis-Trans Questions

To draw cis/trans:

1. Locate C=C.
2. Check each double-bond carbon has two different groups.
3. Put similar groups on same side for cis.
4. Put similar groups on opposite sides for trans.

Boiling point note:

Cis isomer often has higher boiling point due to net dipole.

Melting point note:

Trans isomer often has higher melting point due to better packing.

## 33.6 Aromaticity Questions

Use four conditions:

1. Cyclic.
2. Planar.
3. Fully conjugated.
4. `(4n + 2)` pi electrons.

If any condition fails, the system is not aromatic.

## 33.7 Conversions to Benzene

Common routes:

| Starting Compound | Route |
| --- | --- |
| Ethyne | Cyclic polymerisation: `3HC#CH -> C6H6` |
| Ethene | Convert to ethyne through suitable halogenation/dehydrohalogenation, then cyclise |
| Hexane | Aromatisation/reforming to benzene |
| Sodium benzoate | Soda lime decarboxylation |
| Phenol | Zinc dust reduction |

## 33.8 Electrophilic Substitution Conversion Questions

Examples:

### Benzene to p-nitrobromobenzene

Strategy:

1. Brominate benzene first to form bromobenzene.
2. Bromine is ortho/para directing.
3. Nitrate bromobenzene.
4. Para product is a major product along with ortho.

### Benzene to m-nitrochlorobenzene

Strategy:

1. Nitrate benzene first to form nitrobenzene.
2. Nitro group is meta directing.
3. Chlorinate nitrobenzene.
4. Meta product forms mainly.

### Benzene to p-nitrotoluene

Strategy:

1. Methylate benzene by Friedel-Crafts alkylation to form toluene.
2. Methyl group is ortho/para directing.
3. Nitrate toluene.
4. Para product is major along with ortho.

### Benzene to acetophenone

Strategy:

Use Friedel-Crafts acylation:

```text
C6H6 + CH3COCl --AlCl3--> C6H5COCH3
```

Activating groups increase reactivity.

Deactivating groups decrease reactivity.

Example:

`toluene > benzene > nitrobenzene`

Why?

- `-CH3` activates ring by electron donation.
- benzene is the reference.
- `-NO2` strongly deactivates ring by electron withdrawal.

Common Lewis acid:

`AlCl3`

Other Lewis acids:

- `FeCl3`
- `BF3`
- `SnCl4`

These help generate electrophiles.

# 35. One-Page Revision Summary

Hydrocarbons contain only carbon and hydrogen.

They are classified as alkanes, alkenes, alkynes, and aromatic hydrocarbons.

Alkanes are saturated hydrocarbons with general formula `CnH2n+2`. They contain only sigma bonds and are relatively unreactive. Their important reactions are free radical halogenation, combustion, controlled oxidation, isomerisation, aromatisation, steam reforming, and cracking. Alkanes show conformational isomerism due to rotation around C-C single bonds. Staggered ethane is more stable than eclipsed ethane because torsional strain is lower.

Alkenes contain C=C and have general formula `CnH2n`. The double bond consists of one sigma and one pi bond. The pi bond makes alkenes electron-rich and reactive toward electrophiles. Alkenes show addition reactions, Markovnikov addition, peroxide effect with HBr, oxidation, ozonolysis, and polymerisation. They also show geometrical isomerism because rotation around C=C is restricted.

Alkynes contain C#C and have general formula `CnH2n-2`. The triple bond consists of one sigma and two pi bonds. Ethyne is linear and each carbon is `sp` hybridised. Terminal alkynes are acidic because `sp` carbon has high s-character. Alkynes undergo addition reactions, hydration to carbonyl compounds, and polymerisation. Ethyne can form benzene by cyclic polymerisation.

Aromatic hydrocarbons contain stable delocalised pi systems. Benzene is planar, cyclic, and has six delocalised pi electrons, satisfying Huckel rule. Benzene undergoes electrophilic substitution because substitution preserves aromaticity. Common benzene reactions are nitration, halogenation, sulphonation, Friedel-Crafts alkylation, and Friedel-Crafts acylation. Existing substituents direct incoming groups to ortho/para or meta positions. Benzene and some fused aromatic hydrocarbons are toxic and carcinogenic.

# 36. Final Exam-Ready Points

- Hydrocarbons contain only carbon and hydrogen.
- Alkanes are saturated; alkenes and alkynes are unsaturated.
- Alkanes mainly undergo substitution, not addition.
- Alkenes and alkynes mainly undergo addition because of pi bonds.
- Benzene is unsaturated by formula but aromatic by structure.
- Benzene prefers electrophilic substitution because aromaticity is retained.
- Staggered ethane is more stable than eclipsed ethane due to lower torsional strain.
- Markovnikov rule is explained by carbocation stability.
- Peroxide effect is explained by free radical stability and is shown by HBr.
- Terminal alkynes are acidic due to `sp` hybridisation.
- Aromaticity requires cyclic, planar, fully conjugated structure with `(4n + 2)` pi electrons.
- Activating groups generally direct ortho/para.
- Strong electron-withdrawing groups generally direct meta.
- Halogens are deactivating but ortho/para directing.
- Ozonolysis is used to locate double bonds.
- Wurtz reaction is not preferred for odd-carbon alkanes because mixed products form.
