# Part 1: Why This Appendix Matters

## Core Idea

Physics is not only about formulas. It is also about writing quantities correctly. A correct method can still look wrong if the units, prefixes, symbols, or notation are careless.

For example:

```text
25 cm
```

is correct, but:

```text
25 cms
25 cm.
25 c m
```

are not recommended scientific unit forms.

The purpose of this appendix content is to make your physics writing clean, unambiguous, and standard.

---

## What Learners Usually Underestimate

| Learner Mistake | Why It Matters |
| --- | --- |
| Writing `m/s/s` instead of `m s^-2` | It is less standard and can become confusing in compound units |
| Writing `cms` for centimetres | SI symbols do not take plural `s` |
| Writing `Kg` instead of `kg` | Capital letters change the meaning of unit symbols |
| Writing `1 cm3 = 0.01 m3` | This ignores that the prefix is cubed too |
| Treating `ms^-1` as metre per second | `ms` means millisecond if written without spacing |
| Using `p.e.` inside equations | Abbreviations are not used as symbols in equations |

# Part 2: Conversion Factors

## What Is A Conversion Factor?

A conversion factor is an equality between two ways of measuring the same quantity.

Example:

```text
1 in = 2.54 cm
```

This means one inch and 2.54 centimetres represent the same length.

### How To Use It

If a quantity is given in inches and you need centimetres, multiply by:

```text
2.54 cm / 1 in
```

The unit `in` cancels, leaving `cm`.

---

## Length Conversions

| Conversion | Learner Note |
| --- | --- |
| `1 km = 0.6215 mi` | kilometre to mile |
| `1 mi = 1.609 km` | mile to kilometre |
| `1 m = 1.0936 yd = 3.281 ft = 39.37 in` | metre compared with yard, foot, and inch |
| `1 in = 2.54 cm` | exact standard conversion used often |
| `1 ft = 12 in = 30.48 cm` | useful in pressure and English-unit conversions |
| `1 yd = 3 ft = 91.44 cm` | yard conversion |
| `1 light-year = 9.461 x 10^15 m` | astronomical distance |
| `1 angstrom = 0.1 nm = 10^-10 m` | atomic-scale length |

### Example: Inches To Metres

Convert `10 in` to metres.

```text
10 in x (2.54 cm / 1 in) = 25.4 cm
25.4 cm = 0.254 m
```

Answer:

```text
10 in = 0.254 m
```

# Part 3: Geometry Formulae

## Circle

For a circle of radius `r`:

```text
circumference = 2 pi r
area = pi r^2
```

### Use In Physics

These formulae appear in circular motion, waves on circular paths, rotational motion, and cross-sectional areas.

---

## Sphere

For a sphere of radius `r`:

```text
surface area = 4 pi r^2
volume = (4/3) pi r^3
```

### Use In Physics

These are used in gravitation, electrostatics, pressure, density, fluid mechanics, and thermal physics.

---

## Right Circular Cylinder

For a cylinder of radius `r` and height `h`:

```text
curved surface area = 2 pi r h
total surface area = 2 pi r^2 + 2 pi r h
volume = pi r^2 h
```

### Use In Physics

Useful for pipes, fluid columns, pistons, cylinders, and rods.

---

## Triangle

For a triangle with base `a` and altitude `h`:

```text
area = (1/2) a h
```

For any triangle:

```text
A + B + C = 180 degrees
```

Law of cosines:

```text
c^2 = a^2 + b^2 - 2ab cos C
```

### Use In Physics

Triangle laws are used in vector addition, force resolution, relative velocity, and geometry-based derivations.

---

## Pythagorean Theorem

For a right triangle:

```text
a^2 + b^2 = c^2
```

### Common Physics Use

If perpendicular components of a vector are `Ax` and `Ay`, then:

```text
A = sqrt(Ax^2 + Ay^2)
```

# Part 4: Trigonometry And Algebra Tools

## Basic Trigonometric Ratios

For an angle `theta` in a right triangle:

```text
sin theta = opposite / hypotenuse
cos theta = adjacent / hypotenuse
tan theta = opposite / adjacent
```

Also:

```text
tan theta = sin theta / cos theta
```

---

## Useful Identities

| Identity | Use |
| --- | --- |
| `sin(90 degrees - theta) = cos theta` | complementary angles |
| `cos(90 degrees - theta) = sin theta` | complementary angles |
| `sin^2 theta + cos^2 theta = 1` | simplification |
| `sec^2 theta - tan^2 theta = 1` | trigonometric manipulation |
| `csc^2 theta - cot^2 theta = 1` | trigonometric manipulation |

### Learner Tip

In physics, trigonometry usually enters through resolving vectors into components:

```text
horizontal component = A cos theta
vertical component = A sin theta
```

when `theta` is measured from the horizontal.

---

## Quadratic Formula

If:

```text
ax^2 + bx + c = 0
```

then:

```text
x = [-b +/- sqrt(b^2 - 4ac)] / (2a)
```

### Use In Physics

This appears in projectile motion, kinematics, optics, and any situation where the unknown appears squared.

---

## Binomial Approximation

For small `x`:

```text
(1 - x)^n approximately equals 1 - nx
```

and:

```text
(1 - x)^(-n) approximately equals 1 + nx
```

### Condition

Use these only when:

```text
|x| << 1
```

### Physics Use

Small-height approximations in gravitation and small-extension approximations in mechanics often use binomial expansion.

---

## Exponential Expansion

The source lists the expansion pattern:

```text
e^x = 1 + x + x^2/2! + x^3/3! + ...
```

### Physics Use

This appears in decay, growth, damping, charging circuits, thermal processes, and wave analysis.

# Part 5: Vector Products

## Writing A Vector In Components

A vector `a` with components `ax`, `ay`, and `az` can be written as:

```text
a = ax i + ay j + az k
```

where `i`, `j`, and `k` are unit vectors along the `x`, `y`, and `z` axes.

---

## Dot Product

The dot product of two vectors `a` and `b` is:

```text
a . b = ab cos theta
```

In components:

```text
a . b = ax bx + ay by + az bz
```

### Meaning

The dot product measures how much one vector lies along another.

### Physics Examples

| Formula | Dot Product Meaning |
| --- | --- |
| `W = F . s` | work done by force |
| `P = F . v` | power delivered by force |

---

## Cross Product

The cross product magnitude is:

```text
|a x b| = ab sin theta
```

The direction is perpendicular to the plane containing `a` and `b`, given by the right-hand rule.

### Component Form

```text
a x b =
(ay bz - az by)i
+ (az bx - ax bz)j
+ (ax by - ay bx)k
```

### Physics Examples

| Formula | Cross Product Meaning |
| --- | --- |
| `tau = r x F` | torque |
| `L = r x p` | angular momentum |
| `F = q(v x B)` | magnetic force on a moving charge |

---

## Scalar Triple Product

The source includes:

```text
a . (b x c) = b . (c x a) = c . (a x b)
```

### Meaning

This expression gives the signed volume of a parallelepiped formed by the three vectors.

---

## Vector Triple Product

The source gives:

```text
a x (b x c) = (a . c)b - (a . b)c
```

### Warning

Do not treat cross product like ordinary multiplication. In general:

```text
a x (b x c) is not equal to (a x b) x c
```

# Part 6: SI Derived Units

## What Is A Derived Unit?

A derived unit is built from SI base units.

Example:

```text
force = mass x acceleration
N = kg m s^-2
```

So the newton is a derived unit.

---

## Important Derived Units

| Quantity | SI Unit | In Base Units |
| --- | --- | --- |
| Force | newton, `N` | `kg m s^-2` |
| Pressure | pascal, `Pa` | `N m^-2 = kg m^-1 s^-2` |
| Work/Energy | joule, `J` | `N m = kg m^2 s^-2` |
| Power | watt, `W` | `J s^-1 = kg m^2 s^-3` |
| Charge | coulomb, `C` | `A s` |
| Potential difference | volt, `V` | `W A^-1` |
| Resistance | ohm | `V A^-1` |
| Magnetic flux | weber, `Wb` | `V s` |
| Magnetic field | tesla, `T` | `Wb m^-2` |

### Learner Habit

When confused by a unit, break it into base units. This often reveals whether a formula is dimensionally correct.

# Part 7: Rules For Physical Quantity Symbols

## Quantity Symbols Are Usually Italic

The source says physical quantities are normally single letters printed in italic or sloping type.

Examples:

```text
m = mass
v = velocity
F = force
t = time
```

### Learner Explanation

Quantity symbols represent variables. Their values can change from problem to problem.

---

## Unit Symbols Are Not The Same As Quantity Symbols

The same letter can mean different things depending on style and context.

| Symbol | Could Mean |
| --- | --- |
| `W` italic | work or weight as a physical quantity |
| `W` upright | watt, unit of power |
| `m` italic | mass as a variable in an equation |
| `m` upright | metre, unit of length |
| `g` italic | acceleration due to gravity |
| `g` upright | gram |

### Example

In:

```text
W = m g
```

`W`, `m`, and `g` are physical quantity symbols. They should not be confused with watt, metre, or gram.

---

## Abbreviations Should Not Be Used In Equations

The source warns that abbreviations such as `p.e.` for potential energy are not used as physical symbols in equations.

Better:

```text
U = mgh
```

Avoid:

```text
p.e. = mgh
```

### Reason

Physics equations need standard symbols, not informal text abbreviations.

---

## Vectors

Vectors are commonly printed in bold upright type in textbooks. In handwritten work, an arrow over the symbol is acceptable.

Examples:

```text
vector a
vector F
vector v
```

### Learner Rule

When writing by hand, use arrows for vectors and no arrows for magnitudes:

```text
vector v = velocity vector
v = speed or magnitude of velocity
```

---

## Chemical Elements And Nuclides

Chemical element symbols are written in normal upright type:

```text
Ca, C, H, He, U
```

For a nuclide such as uranium-235:

```text
235U
```

The mass number is written as a left superscript. Atomic number may be written as a left subscript when needed.

For ions, charge is written as a right superscript:

```text
Ca2+
PO4^3-
```

# Part 8: Rules For SI Unit Symbols

## Unit Symbols Are Upright

Symbols for units are written in normal roman upright type, not italic.

Examples:

```text
m, kg, s, A, K, mol, cd
```

---

## Unit Symbols Are Not Abbreviations

This is why unit symbols:

1. do not take plural `s`,
2. do not end with a full stop,
3. remain unchanged in singular and plural.

Correct:

```text
1 cm
25 cm
```

Incorrect:

```text
25 cms
25 cm.
```

---

## Capitalisation Matters

Unit symbols are capitalised only when derived from a proper name.

| Unit | Symbol | Reason |
| --- | --- | --- |
| metre | `m` | not named after a person |
| second | `s` | not named after a person |
| hertz | `Hz` | named after Hertz |
| joule | `J` | named after Joule |
| volt | `V` | named after Volta |
| weber | `Wb` | named after Weber |
| tesla | `T` | named after Tesla |

### Important

`kg` is correct. `Kg` is not correct for kilogram.

---

## The Litre Exception

The source notes that `L` may be used for litre to avoid confusion between lowercase `l` and the numeral `1`.

Correct:

```text
1 L
```

Also common:

```text
1 l
```

But `L` is clearer in many printed contexts.

---

## Solidus Rule

The solidus `/` means division. It should be used carefully.

Acceptable:

```text
m/s^2
m s^-2
J/K mol
J K^-1 mol^-1
```

Avoid:

```text
m/s/s
J/K/mol
kg/m/s
```

### Better Habit

Use negative powers for compound units:

```text
kg m^-1 s^-1
```

instead of:

```text
kg/m/s
```

---

# Part 9: SI Prefixes

## What A Prefix Does

A prefix multiplies a unit by a power of ten.

Examples:

```text
1 km = 10^3 m
1 cm = 10^-2 m
1 ns = 10^-9 s
1 MW = 10^6 W
```

The prefix symbol is written directly next to the unit symbol with no space.

---

## Common Prefixes

| Prefix | Symbol | Factor |
| --- | --- | --- |
| giga | `G` | `10^9` |
| mega | `M` | `10^6` |
| kilo | `k` | `10^3` |
| centi | `c` | `10^-2` |
| milli | `m` | `10^-3` |
| micro | `micro` | `10^-6` |
| nano | `n` | `10^-9` |
| pico | `p` | `10^-12` |
| femto | `f` | `10^-15` |

Note: the raw source uses the standard micro symbol, but this learner file writes `micro` in plain text to keep the file ASCII-clean.

---

## Prefixes Are Attached To Units

Correct:

```text
km
cm
ns
MW
```

Incorrect:

```text
k m
c m
n s
M W
```

The prefix has no independent physical meaning unless attached to a unit.

---

## Powers Apply To The Whole Prefixed Unit

This is one of the most important rules in the appendix.

```text
cm^3 means (cm)^3
```

So:

```text
1 cm^3 = (10^-2 m)^3 = 10^-6 m^3
```

It does not mean:

```text
10^-2 m^3
```

### More Examples

```text
1 km^2 = (10^3 m)^2 = 10^6 m^2
1 mm^2 = (10^-3 m)^2 = 10^-6 m^2
1 cm^-1 = (10^-2 m)^-1 = 10^2 m^-1
```

---

## Avoid Double Prefixes

Use a single available prefix instead of combining two prefixes.

Correct:

```text
1 nm = 10^-9 m
1 pF = 10^-12 F
1 GW = 10^9 W
```

Avoid:

```text
millimicrometre
micromicrofarad
kilomegawatt
```

---

## Prefix In Numerator Is Preferred

The source notes that it is often clearer to attach prefixes to units in the numerator.

Better:

```text
MN/m^2
```

than:

```text
N/mm^2
```

This avoids confusion when powers and prefixes combine.

# Part 10: Spacing Rules That Prevent Confusion

## `m s^-1` Versus `ms^-1`

These do not mean the same thing.

```text
m s^-1
```

means metre per second.

```text
ms^-1
```

means per millisecond, because `ms` is millisecond.

### Learner Rule

Use spacing between multiplied unit symbols:

```text
m s^-1
N m
C m
```

Do not accidentally join unit symbols into a prefix-unit combination.

---

## `C m` Versus `cm`

```text
C m
```

means coulomb metre.

```text
cm
```

means centimetre.

This is why spacing is not decorative. It changes meaning.

---

## `mS` Versus `ms`

```text
mS
```

means millisiemens.

```text
ms
```

means millisecond.

Capitalisation and spacing both matter.

---

# Part 11: Mass Prefix Exception

## Why Kilogram Is Special

The SI base unit for mass is kilogram, but prefixes for smaller and larger mass units are attached to gram, not kilogram.

Correct:

```text
1 mg = 10^-6 kg
1 g = 10^-3 kg
1 Mg = 10^3 kg
```

Avoid:

```text
1 microkilogram
1 millikilogram
1 kilokilogram
```

### Learner Explanation

This looks unusual because kilogram already contains a prefix, `kilo`. Historical reasons created this exception, so standard SI usage attaches prefixes to gram for mass multiples and submultiples.

# Part 12: Applying The Appendix To Fluid Mechanics

Even though the raw source is appendix material, these rules are especially important in fluid mechanics.

## Pressure

Fluid pressure is:

```text
P = F/A
```

SI unit:

```text
Pa = N m^-2
```

If a problem gives pressure in atmosphere, bar, torr, or mm Hg, convert to pascal before using SI equations.

---

## Density

Density is:

```text
rho = m/V
```

Common conversion:

```text
1 g cm^-3 = 1000 kg m^-3
```

Water has density close to:

```text
1000 kg m^-3
```

---

## Volume

Fluid volume may appear in litres:

```text
1 L = 10^-3 m^3
```

Before using equations such as continuity or pressure-volume work, convert litres to cubic metres.

---

## Pressure-Volume Work

The source gives:

```text
1 L atm = 101.325 J
```

This matters because pressure times volume has units of energy:

```text
Pa m^3 = (N m^-2)m^3 = N m = J
```

# Part 13: Common Exam Errors And Fixes

## Error 1: Forgetting To Square Or Cube The Prefix

Wrong:

```text

```

Correct:

```text
1 cm^3 = (10^-2 m)^3 = 10^-6 m^3
```

---

## Error 2: Pluralising Unit Symbols

Wrong:

```text
10 kgs
25 cms
5 mins
```

Correct:

```text
10 kg
25 cm
5 min
```

---

## Error 3: Confusing Quantity Symbol With Unit Symbol

Wrong explanation:

```text
W is always watt.
```

Correct explanation:

```text
W may be a physical quantity symbol in an equation, but W as a unit symbol means watt.
Context and typography matter.
```

---

## Error 4: Using Informal Abbreviations In Equations

Wrong:

```text
p.e. = mgh
```

Better:

```text
U = mgh
```

---

## Error 5: Misreading Joined Unit Symbols

Wrong:

```text
ms^-1 means metre per second.
```

Correct:

```text
m s^-1 means metre per second.
ms^-1 means per millisecond.
```

# Final Revision Sheet

## Must Remember

- Unit symbols are standard symbols, not abbreviations.
- Unit symbols do not take plural endings.
- Prefix symbols attach directly to unit symbols.
- A prefixed unit raised to a power means the prefix is raised too.
- Do not use double prefixes.
- Use spacing between multiplied unit symbols.
- `m s^-1` means metre per second; `ms^-1` means per millisecond.
- Physical quantities are variables; units are measurement standards.
- Pressure unit `Pa` equals `N m^-2`.
- Energy unit `J` equals `N m`.
- `1 L = 10^-3 m^3`.
- `1 g cm^-3 = 1000 kg m^-3`.
- `1 kW h = 3.6 MJ`.
- `1 eV = 1.602 x 10^-19 J`.
- Dot product gives scalar projection-type results, such as work.
- Cross product gives perpendicular vector results, such as torque.
