# Chapter 4: Moving Charges and Magnetism

Moving charges produce magnetic effects and experience magnetic forces. This chapter studies Lorentz force, motion of charged particles in magnetic fields, magnetic field due to currents, Biot-Savart law, Ampere's circuital law, solenoids, force between parallel currents, torque on current loops, and moving coil galvanometer.

The central idea is that magnetism is linked with moving charge.

# Unit 4A: Magnetic Field and Moving Charge

A charge at rest does not experience magnetic force. A moving charge in a magnetic field experiences force if its velocity has a component perpendicular to the magnetic field.

Magnetic force:

`F = q v B sin theta`

where `theta` is angle between velocity and magnetic field.

## Direction

Direction is given by vector product `v x B` for positive charge. For negative charge, direction is opposite.

# Unit 4B: Lorentz Force

If both electric and magnetic fields are present:

`F = q(E + v x B)`

Electric force can do work and change speed. Magnetic force is perpendicular to velocity, so it does no work and changes direction of motion.

## Important Result

Magnetic force alone cannot change kinetic energy of a charged particle.

# Unit 4C: Force on Current-Carrying Conductor

A current-carrying wire in a magnetic field experiences force.

`F = I L B sin theta`

where `L` is length vector in direction of current.

## Direction

Use right-hand rule for `L x B`.

## Special Cases

- If wire is parallel to field: `theta = 0`, force is zero.
- If wire is perpendicular to field: `theta = 90 degrees`, force is maximum, `F = ILB`.

# Unit 4D: Motion in a Uniform Magnetic Field

If a charged particle enters a uniform magnetic field perpendicular to its velocity, it moves in a circle.

Magnetic force provides centripetal force:

`qvB = mv^2/r`

Therefore:

`r = mv/(|q|B)`

Angular frequency:

`omega = |q|B/m`

Time period:

`T = 2 pi m/(|q|B)`

## If Velocity Is Oblique

If velocity has components parallel and perpendicular to magnetic field:

- Perpendicular component gives circular motion.
- Parallel component remains unchanged.
- Result is helical motion.

# Unit 4E: Biot-Savart Law

Biot-Savart law gives magnetic field due to a current element.

For current element `I dl`, magnetic field at a point is proportional to:

- Current `I`.
- Element length `dl`.
- `sin theta`.

and inversely proportional to square of distance.

## Vector Meaning

Direction of magnetic field is perpendicular to plane containing current element and position vector, given by right-hand rule.

# Unit 4F: Field Due to Long Straight Wire

For a long straight wire carrying current `I`, magnetic field at distance `r` is:

`B = mu0 I/(2 pi r)`

Field lines are concentric circles around the wire.

## Right-Hand Thumb Rule

Point thumb in direction of current. Curled fingers show direction of magnetic field.

# Unit 4G: Field on Axis of a Circular Current Loop

For a circular loop of radius `R` carrying current `I`, magnetic field at centre is:

`B = mu0 I/(2R)`

For `N` turns:

`B = mu0 N I/(2R)`

On the axis at distance `x` from centre:

`B = mu0 I R^2/[2(R^2+x^2)^(3/2)]`

For `N` turns, multiply by `N`.

# Unit 4H: Ampere's Circuital Law

Ampere's circuital law states:

`line integral of B.dl over closed path = mu0 times current enclosed`

Symbolically:

`integral B.dl = mu0 I_enclosed`

## Use

Ampere's law is useful when symmetry is high:

- Long straight wire.
- Infinite current sheet.
- Long solenoid.
- Toroid.

# Unit 4I: Long Solenoid

A solenoid is a long coil of many turns.

Inside a long ideal solenoid:

`B = mu0 n I`

where `n` is number of turns per unit length.

Outside a long ideal solenoid, magnetic field is nearly zero.

## With Core

If a magnetic material is inserted, field changes depending on magnetic permeability of the material.

# Unit 4J: Force Between Two Parallel Currents

Two long parallel wires carrying currents exert force on each other.

Force per unit length:

`F/L = mu0 I1 I2/(2 pi d)`

where `d` is separation.

## Direction

- Currents in same direction attract.
- Currents in opposite directions repel.

## Definition of Ampere

This force law historically helped define ampere in SI.

# Unit 4K: Torque on a Current Loop

A rectangular current loop in a uniform magnetic field experiences torque.

`tau = N I A B sin theta`

where:

- `N` = number of turns.
- `I` = current.
- `A` = area.
- `theta` = angle between magnetic moment and magnetic field.

Magnetic moment:

`m = N I A`

Vector form:

`tau = m x B`

# Unit 4L: Current Loop as Magnetic Dipole

A current loop behaves like a magnetic dipole.

Magnetic dipole moment:

`m = NIA`

Direction is normal to the plane of the loop, given by right-hand rule.

## Analogy

Current loop in magnetic field behaves like electric dipole in electric field:

- Electric dipole torque: `p x E`
- Magnetic dipole torque: `m x B`

# Unit 4M: Moving Coil Galvanometer

A moving coil galvanometer works on torque experienced by a current loop in magnetic field.

Torque due to magnetic field:

`tau = N I A B`

for radial magnetic field, where `sin theta = 1`.

Restoring torque from spring:

`tau = k phi`

At equilibrium:

`N I A B = k phi`

So:

`phi proportional to I`

This makes galvanometer a current-measuring device.

# Unit 4N: Worked Examples

## Example 1: Magnetic Force on Charge

A charge `2 microC` moves with speed `3 x 10^5 m/s` perpendicular to `0.5 T` field.

`F = qvB`

`F = (2 x 10^-6)(3 x 10^5)(0.5)`

`F = 0.3 N`

## Example 2: Radius of Circular Path

For particle of mass `m`, charge `q`, speed `v`, in field `B`:

`r = mv/(qB)`

Use magnitude of charge. Direction of charge decides rotation direction, not radius sign.

## Example 3: Field Due to Wire

Long wire carries `10 A`. Find field at `0.05 m`.

`B = mu0 I/(2 pi r)`

Using `mu0 = 4 pi x 10^-7`:

`B = (4 pi x 10^-7)(10)/(2 pi)(0.05)`

`B = 4 x 10^-5 T`

# Unit 4O: Answer-Writing Framework

## To solve magnetic force problems

Write in this order:

1. Identify charge/current, velocity/length, magnetic field, and angle.
2. Use `F = qvB sin theta` or `F = ILB sin theta`.
3. Use right-hand rule for direction.
4. Reverse direction for negative charge if needed.

## To solve circular motion in magnetic field

Write in this order:

1. Set magnetic force equal to centripetal force.
2. Use `qvB = mv^2/r`.
3. Solve for `r`, `T`, or `omega`.
4. Mention that magnetic force does no work.

## To use Ampere's law

Write in this order:

1. Choose Amperian loop matching symmetry.
2. Identify enclosed current.
3. Evaluate `integral B.dl`.
4. Apply `integral B.dl = mu0 I_enclosed`.
5. Solve for `B`.

# Unit 4P: Common Mistakes and Corrections

| Mistake | Correction |
|---|---|
| Applying magnetic force to charge at rest | Magnetic force needs motion. |
| Forgetting `sin theta` | Force depends on angle between velocity/current and field. |
| Saying magnetic force changes speed | Magnetic force alone changes direction, not speed. |
| Ignoring charge sign in direction | Negative charge force is opposite to `v x B`. |
| Using straight-wire formula for finite wire without care | `B = mu0I/(2pi r)` is for long straight wire. |
| Forgetting `N` turns in coil formulas | Multiply field or torque by number of turns. |
| Confusing current direction and electron motion | Conventional current direction is used in formulas. |

## Final Revision

Remember these points:

- Moving charge in magnetic field experiences force `qvB sin theta`.
- Magnetic force is perpendicular to velocity.
- A perpendicular magnetic field causes circular motion.
- Currents create magnetic fields.
- Ampere's law is powerful for symmetric current distributions.
- Parallel currents attract if same direction and repel if opposite.
- Current loop behaves as a magnetic dipole.
