# Unit 5A: Structure of DNA

DNA is a polymer of nucleotides. Each nucleotide contains:

1. Deoxyribose sugar.
2. Phosphate group.
3. Nitrogenous base.

## Bases in DNA

| Purines | Pyrimidines |
|---|---|
| Adenine, guanine | Cytosine, thymine |

Adenine pairs with thymine using two hydrogen bonds. Guanine pairs with cytosine using three hydrogen bonds.

## Double Helix Features

1. DNA has two antiparallel strands.
2. Sugar-phosphate backbone is outside.
3. Bases face inward.
4. Base pairing is complementary.
5. Complementarity allows accurate replication.

# Unit 5B: Structure of RNA

RNA usually has a single strand and ribose sugar.

## Bases in RNA

RNA has adenine, guanine, cytosine, and uracil. Uracil replaces thymine.

## Types of RNA

| RNA Type | Function |
|---|---|
| mRNA | Carries genetic message from DNA to ribosome. |
| tRNA | Brings amino acids during translation. |
| rRNA | Structural and catalytic component of ribosome. |

# Unit 5C: DNA as Genetic Material

Genetic material should:

1. Replicate accurately.
2. Store information.
3. Express information as traits.
4. Mutate to create variation.
5. Be chemically stable enough for inheritance.

DNA satisfies these requirements in most organisms.

## Transforming Principle

Experiments with bacteria showed that hereditary information could transform one bacterial type into another. Later work identified DNA as the transforming substance.

## Hershey-Chase Experiment

Bacteriophages were used to show that DNA enters bacterial cells and directs viral multiplication, while protein coat remains outside. This supported DNA as genetic material.

# Unit 5D: RNA World

RNA can store information and catalyse reactions in some systems. This supports the idea that early life may have used RNA before DNA and proteins became dominant.

RNA is less stable than DNA, which may be why DNA became the main genetic material in most organisms.

# Unit 5E: DNA Replication

DNA replication is semi-conservative. Each new DNA molecule contains one parental strand and one newly synthesised strand.

## Key Enzymes and Proteins

| Component | Function |
|---|---|
| Helicase | Unwinds DNA helix. |
| DNA polymerase | Adds nucleotides to growing strand. |
| Primase | Makes RNA primer. |
| Ligase | Joins DNA fragments. |
| Topoisomerase | Reduces supercoiling. |

## Direction

DNA polymerase adds nucleotides only in 5' to 3' direction. Therefore:

1. Leading strand is synthesised continuously.
2. Lagging strand is synthesised discontinuously as Okazaki fragments.

## Steps

1. Origin of replication opens.
2. Helicase unwinds DNA.
3. Primers are formed.
4. DNA polymerase extends new strands.
5. Primers are replaced.
6. Ligase seals gaps.

# Unit 5F: Transcription

Transcription is synthesis of RNA from DNA template.

## Basic Requirements

1. DNA template.
2. RNA polymerase.
3. Ribonucleotides.
4. Promoter.
5. Terminator.

## DNA Strands

| Strand | Role |
|---|---|
| Template strand | Read by RNA polymerase to make RNA. |
| Coding strand | Has sequence similar to RNA, except T instead of U. |

## Transcription Unit

A transcription unit includes:

1. Promoter.
2. Structural gene.
3. Terminator.

# Unit 5G: RNA Processing in Eukaryotes

In eukaryotes, primary RNA transcript is processed before translation.

## Processing Steps

| Step | Meaning |
|---|---|
| Capping | Modified nucleotide added at 5' end. |
| Tailing | Adenine-rich tail added at 3' end. |
| Splicing | Introns removed and exons joined. |

Introns are non-coding sequences removed during processing. Exons are expressed sequences retained in mature mRNA.

# Unit 5H: Genetic Code

Genetic code is the relationship between mRNA codons and amino acids.

## Features

1. Triplet: three bases form one codon.
2. Universal: mostly same across organisms.
3. Degenerate: more than one codon may code for same amino acid.
4. Unambiguous: one codon specifies only one amino acid.
5. Non-overlapping: codons are read sequentially.
6. Commaless: no punctuation between codons.
7. AUG codes methionine and acts as start codon.
8. UAA, UAG, and UGA are stop codons.

# Unit 5I: Translation

Translation is protein synthesis from mRNA.

## Main Components

1. mRNA.
2. tRNA.
3. Ribosomes.
4. Amino acids.
5. Enzymes and energy.

## tRNA Role

tRNA has:

1. Anticodon loop to recognise codon.
2. Amino acid attachment site.

## Steps

1. Activation of amino acids.
2. Initiation at start codon.
3. Elongation by adding amino acids.
4. Termination at stop codon.
5. Release of polypeptide.

# Unit 5J: Regulation of Gene Expression

Not all genes are active all the time. Gene regulation allows cells to produce proteins when needed.

## Lac Operon

The lac operon in bacteria controls lactose metabolism.

Components:

1. Regulator gene.
2. Promoter.
3. Operator.
4. Structural genes.

## Without Lactose

Repressor binds operator and blocks transcription.

## With Lactose

Lactose or its isomer binds repressor, preventing it from binding operator. Structural genes are transcribed.

This is inducible regulation.

# Unit 5K: Mutation

Mutation is a change in DNA sequence.

## Types

1. Point mutation: change in one base pair.
2. Frameshift mutation: insertion or deletion changes reading frame.
3. Chromosomal mutation: change in chromosome structure.

## Effects

Mutation may:

1. Have no effect.
2. Change one amino acid.
3. Create stop codon.
4. Shift reading frame.
5. Affect gene regulation.

# Unit 5L: Human Genome Programme

The human genome project aimed to identify and map all genes in human DNA and determine nucleotide sequence.

## Importance

1. Improved understanding of genetic disorders.
2. Helped gene mapping.
3. Supported medical genetics.
4. Provided data for comparative genomics.
5. Improved forensic and evolutionary studies.

# Unit 5M: DNA Fingerprinting

DNA fingerprinting identifies individuals using differences in DNA sequences.

## Basis

Individuals differ in repeated DNA sequences such as VNTRs. The number and arrangement of repeats vary among individuals.

## Uses

1. Forensic identification.
2. Paternity and maternity testing.
3. Identification of remains.
4. Population and evolutionary studies.
5. Conservation genetics.

## Main Steps

1. Isolate DNA.
2. Cut DNA or amplify target regions.
3. Separate fragments.
4. Transfer or detect DNA pattern.
5. Compare banding or sequence profile.

# Unit 5N: Common Mistakes and Corrections

| Mistake | Correct Understanding |
|---|---|
| DNA strands are parallel | They are antiparallel. |
| RNA has thymine | RNA has uracil instead of thymine. |
| Both DNA strands are transcribed for one gene | Usually one strand acts as template for a gene. |
| Replication is conservative | It is semi-conservative. |
| All mutations are visible | Many are silent or not expressed. |
| Introns code for protein | Introns are removed during RNA processing. |
| Codon is on tRNA | Codon is on mRNA; anticodon is on tRNA. |

## Final Revision

Molecular inheritance follows an information pathway: DNA stores, DNA replicates, DNA is transcribed into RNA, and RNA is translated into protein. Most questions become easier when you identify template, product, enzyme, and direction for each process.
